L01 全部は調べられない——全数調査と標本調査

準備運動（前提診断）

  1. 3の目が出る確率は 1/6。「同様に確からしい」＝どの目が出る確率も等しいということ（1〜6のどの目も1/6）。

  2. データは8個（小さい順に 2, 3, 5, 6, 8, 9, 11, 12）。 中央値＝4番目と5番目の平均＝(6＋8)÷2＝7。 第1四分位数＝下位4個の中央値＝(3＋5)÷2＝4、第3四分位数＝上位4個の中央値＝(9＋11)÷2＝10。 四分位範囲＝10−4＝6（検算: 10＞4で正の値 ✓）

  3. 14÷40＝0.35（検算: 40×0.35＝14 ✓）

練習

  1. (ア) 全数調査——一人ひとり（各世帯）の情報そのものが必要な国の基本調査だから。 (イ) 標本調査——全部開封すると売る物がなくなるから。 (ウ) 全数調査——各自の進路希望そのものが必要な情報だから。 (エ) 標本調査——県の中学生全員を調べるのは手間・時間がかかりすぎ、全体の傾向が分かれば目的を果たせるから。

  2. 寿命を調べるには電池を使い切るまで試すことになり、調べた電池は商品として使えなくなる。全数調査にすると売る製品がすべてなくなってしまう（調べる行為が対象をこわす・なくす型の典型）。

  3. 例: 対象が非常に多い調査では、全数調査は手間・時間・費用がかかりすぎ、調べもれや集計ミスが入りこむ余地も大きくなる。目的（全体の傾向を知る）に対しては、きちんと設計した標本調査の方が現実的で優れていることがある。（缶詰の開封検査のように「全数調査がそもそも成立しない」例を挙げてもよい。）

stretch

S1 一例: テレビ番組についての意識調査。(a) 視聴者全員に聞くことは人数的に不可能で、全体の傾向が分かれば目的を果たせるから。(b) 全体＝その調査が語ろうとしている人々の集まり（例: 全国の世帯全体）。※答えは1つではない。「(a)全数調査にしない理由」「(b)全体が何か」の2点が書けていればよい。

L02 公平に選ぶ——母集団・標本・無作為抽出

練習

  1. 母集団＝さくら市の中学生9,000人、標本＝選ばれた300人、標本の大きさ＝300。

  2. 無作為抽出といえるのは （イ）（名簿の全員に番号がつき、乱数でどの生徒も等しい確率で選ばれる）。 (ア) 図書室に来る生徒は本をよく読む側にかたよるおそれがある（読書量の調査としては特にまずい選び方）。 (ウ) 1つの学級だけでは、学年（3年生のみ）や学級の雰囲気など特定の顔ぶれにかたよるおそれがある。

  3. 001≦番号≦320 の数だけを順に採用する。 483（範囲外・とばす）→ 072 → 615（とばす）→ 129 → 350（とばす）→ 288 → 904（とばす）→ 017 → 460（とばす）→ 133 選ばれる5人の番号は 072, 129, 288, 017, 133。（検算: 5個とも 001≦番号≦320 の範囲内・重複なし ✓）

  4. 「くわしい友だち5人」は選ぶ人の判断で選ばれており、母集団のどの生徒も等しい確率で選ばれるという条件を満たさない。くわしい人・関心の強い人にかたよった標本になり、母集団全体のようす（くわしくない人もふくめた全体像）を知る目的には合わない。

stretch

S1 方法の例: 表計算ソフトで「1以上320以下の整数の一様乱数」を作る関数を使う（具体的な関数名は使う道具により異なるので、調べて確かめる）。同じ操作をもう一度すると、出てくる10個はふつう前回と異なる——乱数は毎回新しく作られるからで、これが「標本はそのたびに顔が違う」（L03）の入り口になる。

L03 標本はそのたびに顔が違う——標本平均のばらつき

練習

  1. 標本B: 合計＝7.5＋7.0＋8.0＋7.5＋7.0＋6.5＋7.5＋7.0＋7.5＋7.5＝73.0 → 73.0÷10＝7.30（時間） 標本C: 合計＝6.0＋7.0＋7.0＋6.5＋7.5＋7.0＋6.5＋7.0＋7.5＋7.0＝69.0 → 69.0÷10＝6.90（時間） （検算: 平均×10がそれぞれ合計73.0・69.0に戻る ✓） 数直線には 6.90（C）・7.10（A）・7.30（B）の3点を打つ。3点は1か所に重ならず散らばる。

  2. 正しいのは （イ）。 (ア) 無作為抽出を正しく行っても、標本の平均値は標本ごとにばらつく。値がちがうこと自体は手順のまちがいの証拠ではない。 (ウ) どの標本の平均値が母集団の平均値に近いかは、この3つの値だけからは判断できない（「大きいほど近い」という根拠はない）。

  3. 例: 3つの平均値の平均 (7.10＋7.30＋6.90)÷3＝7.10 も、標本から作った値である以上ばらつきをもつ見積もりであり、「正確に求められる」とは言えない。言えるのは「母集団の平均値はおよそ7.1時間と推定される」まで。（「正確に」という断定が問題であることが指摘できていればよい。）

stretch

S1 設問の約束どおり、3回の抽出で同じ生徒の重複はない（重複したら選び直した）ものとする。30人分の合計＝71.0＋73.0＋69.0＝213.0 → 213.0÷30＝7.10（時間）。 印象の例: 大きさ10の3つの値（6.90〜7.30）がばらついていたのに対し、まとめて大きさ30にした値は、その散らばりの中ほどに落ち着いた感じがする——「標本を大きくすると値が安定しそうだ」という予感が、L04のテーマになる。

L04 標本を大きくすると——箱ひげ図でばらつきを見る

練習

  1. 標本の大きさ20の記録（20個を小さい順に並べる）: 6.8, 6.9, 7.0, 7.0, 7.1, 7.1, 7.1, 7.2, 7.2, 7.2, 7.2, 7.2, 7.3, 7.3, 7.4, 7.4, 7.4, 7.5, 7.5, 7.6

    • 最小値 6.8・最大値 7.6 → 範囲＝7.6−6.8＝0.8

    • 中央値＝(10番目＋11番目)÷2＝(7.2＋7.2)÷2＝7.2

    • 第1四分位数＝(5番目＋6番目)÷2＝(7.1＋7.1)÷2＝7.1、第3四分位数＝(15番目＋16番目)÷2＝(7.4＋7.4)÷2＝7.4 → 四分位範囲＝7.4−7.1＝0.3

    標本の大きさ40の記録（小さい順）: 7.0, 7.0, 7.1, 7.1, 7.1, 7.1, 7.2, 7.2, 7.2, 7.2, 7.2, 7.2, 7.2, 7.3, 7.3, 7.3, 7.3, 7.4, 7.4, 7.4

    • 最小値 7.0・最大値 7.4 → 範囲＝0.4

    • 中央値＝(7.2＋7.2)÷2＝7.2

    • 第1四分位数＝(7.1＋7.1)÷2＝7.1、第3四分位数＝(7.3＋7.3)÷2＝7.3 → 四分位範囲＝7.3−7.1＝0.2

    （検算: 並べかえ後の個数がどちらも20個であること、最小値≦第1四分位数≦中央値≦第3四分位数≦最大値の順になっていることを確認 ✓） 箱ひげ図は、大きさ20が「ひげ6.8〜7.6・箱7.1〜7.4・中央線7.2」、大きさ40が「ひげ7.0〜7.4・箱7.1〜7.3・中央線7.2」となる。

  2. 表の例:

    標本の大きさ範囲四分位範囲
    101.50.4
    200.80.3
    400.40.2

    記述例: 「標本の大きさを大きくすると、標本の平均値の範囲・四分位範囲は小さくなる傾向がある。」（「必ず小さくなる」と書かないこと。）

  3. 大きさ40の記録にも、たとえば 7.0（または7.4）のように中央の7.2から離れた値が残っている。つまり大きさ40でも標本の平均値はゆれており、「必ずほぼ一致する」は言いすぎ。言えるのは「ばらつきが小さくなる傾向がある」まで。

  4. 例: 調べる手間・時間・費用が増える。標本を大きくするほど全数調査に近づき、「全部は調べられないから標本調査にした」というそもそもの利点が失われていく。（回答が1つの観点で書けていればよい。）

stretch

S1 大きさ10の記録で7.0以上7.4以下の値: 7.0, 7.0, 7.1, 7.1, 7.2, 7.2, 7.2, 7.3, 7.3, 7.4, 7.4, 7.4 の 12個（12/20＝6割）。大きさ40の記録では20個全部（20/20）。 まとめの例: 「中央値のまわり±0.2時間に収まった割合は、標本が大きいほど高い——標本を大きくすると、推定が安定する傾向がここでも確かめられる。」

やってみよう（必須）の確かめ

この活動の自己採点は、値が下の参照例と一致するかではなく、手順が正しく実行できたかで行う。まず次のチェックリストを確かめよう（さいころやコンピュータの乱数を使った人は、番号も平均値も参照例と一致しなくてよい。チェックリストだけで採点する）。

  • □ 【抽出】乱数リストを左上から右へ、行の終わりで次の行への順に読んだ。

  • □ 【抽出】00と61〜99を読み飛ばし、01〜60だけを採用した。

  • □ 【抽出】同じ標本の中で同じ番号が2回出たとき、2回目を読み飛ばした。

  • □ 【抽出】2組目はリストの続きから読み、1組目と重複する番号はそのまま採用した（読み飛ばすのは「同じ標本の中」の重複だけ）。

  • □ 【観測】採用した番号を母集団リストで引き、2組とも10人分の時間を正しく書き出した（それぞれ10個あるか個数を確認）。

  • □ 【整理】2組とも、平均値＝合計÷10 を計算し、平均値×10 が合計に戻ることを検算した。

  • □ 【推定】答えを「およそ□時間」の形（断定しない言い方）で書いた。

  • □ 【比べる】2つの標本の平均値を並べて比べた。

  • □ 【比べる】全数調査の平均値（自分で計算するか、下の答え合わせで確認）と照らした。

付属の乱数リストをそのまま使った場合の参照例（読み方が合っているかの確認用）:

  • 1組目の読みとり: 57採用 → 62飛 → 38 → 25 → 47 → 72飛 → 20 → 43 → 18 → 51 → 69飛 → 91飛 → 49 → 43飛（同じ標本内で2回目）→ 51飛（同）→ 60採用、で10人。 番号: 57, 38, 25, 47, 20, 43, 18, 51, 49, 60 時間: 6.0, 7.0, 6.5, 7.5, 7.0, 7.5, 7.0, 7.0, 7.0, 8.5 → 合計 71.0 → 平均 7.10（時間）（検算: 7.10×10＝71.0 ✓）

  • 2組目の読みとり（続き・2行目の「00」から）: 00飛 → 46採用 → 69飛 → 66飛 → 09 → 12 → 02 → 95飛 → 82飛 → 58 → 14 → 92飛 → 85飛 → 28 → 81飛 → 67飛 → 42 → 43採用（1組目にも入っていたが、別の抽出なので採用する）→ 87飛 → 83飛 → 17採用、で10人。 番号: 46, 09, 12, 02, 58, 14, 28, 42, 43, 17 時間: 5.5, 7.0, 7.5, 8.0, 6.5, 6.0, 7.5, 6.0, 7.5, 7.5 → 合計 69.0 → 平均 6.90（時間）（検算: 6.90×10＝69.0 ✓）

  • 推定の言い方の例: 「ひばり台中1年60人の昨夜の睡眠時間の平均値は、およそ7時間と考えられる。」

全数調査での答え合わせ（この母集団は60人しかいないから、本当の値を確かめられる）:

  • 60人の合計＝420.0時間 → 本当の平均値＝420.0÷60＝7.00（時間）（検算: 7.00×60＝420.0 ✓）

  • 参考（母集団の散らばり・中2の方法で）: 最小値5.5・第1四分位数6.5・中央値7.0・第3四分位数7.5・最大値8.5（四分位範囲1.0・範囲3.0）。

  • 2つの標本平均（7.10と6.90）は、どちらも本当の値7.00とぴたりとは一致しないが、その近くに来ている。「標本の平均値はばらつく（L03）」「それでも母集団の値の近くに集まる（L04）」が、自分の手の中で再現されたことになる。

  • 参考（付属の擬似乱数リストの性質）: 50個の平均は49.68、最小0・第1四分位数28・中央値50.0・第3四分位数69・最大99で、00〜99の全体に大きな偏りは見られない（教材側で検算済み）。ただしこれは「どの数も等しい確率で現れるように生成された」ことの証明ではない。乱数の性質は、結果の並びだけでなく生成のしかたで決まる（このリストはコンピュータの一様乱数から生成した）。
