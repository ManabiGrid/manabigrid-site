L05 平均から全体を見積もる——平均値×総数型

練習

  1. (1) 順にたすと 58＋63＝121, ＋60＝181, ＋59＝240, ＋65＝305, ＋62＝367, ＋57＝424, ＋64＝488, ＋61＝549, ＋60＝609, ＋63＝672, ＋58＝730, ＋62＝792, ＋66＝858, ＋59＝917, ＋61＝978, ＋60＝1038, ＋64＝1102, ＋57＝1159, ＋65＝1224 ✓ (2) 1224÷20＝61.2（語/ページ）、61.2×980＝59976（語）。 （検算: 61.2×980＝61.2×1000−61.2×20＝61200−1224＝59976 ✓。また 59976÷980＝61.2 に戻る ✓） 答えの書き方: 「見出し語はおよそ60000語（およそ6万語）と推定される」。

  2. 8本の合計＝82＋78＋85＋80＋74＋88＋79＋86＝652（個）。標本の平均値＝652÷8＝81.5（個/本）。 500本全体では 81.5×500＝40750（個）。 答え: 「果樹園全体の収穫個数はおよそ41000個（およそ4万個）と推定される」。 （検算: 40750÷500＝81.5 に戻る ✓。合計の確かめ: 82＋78＝160, ＋85＝245, ＋80＝325, ＋74＝399, ＋88＝487, ＋79＝566, ＋86＝652 ✓）

  3. 絵や図の多いページは、紙面が絵に使われるぶん見出し語数が少なくなると考えるのが自然。そのようなページばかり選ぶと標本の平均値は実際より小さくなり、推定値（平均×980）も小さい方にかたよると考えられる。（「にぎやかなページ＝語数が多い」と設定して「大きい方にかたよる」と論じた場合も、標本の選び方のくせが平均値を通じて推定全体をかたよらせる、という筋が通っていれば可。）

  4. 例: 標本の平均値は標本ごとにばらつく値であり（L03）、61.2はたまたま選ばれた20ページでの値にすぎない。別の20ページなら別の値になるから、母集団の平均値が「ぴったり61.2」と言うことはできない。言えるのは「およそ61語くらいと推定される」まで（L04で見たとおり、標本を大きくすればこの見積もりのばらつきは小さくなる傾向がある）。

stretch

S1 例: 入り口に近い木は、人の目が届きやすく世話が行き届いて収穫が多めかもしれない（あるいは、排気ガスや踏み固めで育ちが悪く少なめかもしれない）。どちらの筋でも、「木の並び（場所）と収穫個数に関係がありそうなのに、場所のかたよった8本を選んでいる」という問題が書けていればよい。

L06 割合から全体を見積もる——比例配分型

練習

  1. 標本の割合＝21÷60＝0.35。全校では 480×0.35＝168（人）。 答え: 「自転車通学の生徒はおよそ170人（168人程度）と推定される」。 （検算: 168÷480＝0.35＝標本の割合 ✓）

  2. 標本の割合＝6÷300＝0.02。全体では 25000×0.02＝500（個）。 答え: 「不良品はおよそ500個と推定される」。 （検算: 500÷25000＝0.02＝標本の割合 ✓）

  3. 湖の魚の総数を x 匹とする。分子どうし＝しるしのついた魚、分母どうし＝魚全体をそろえて、 90/x ＝ 5/60 5x ＝ 90×60 ＝ 5400 → x ＝ 1080 答え: 「湖の魚はおよそ1080匹（およそ1100匹）と推定される」。 （検算: 90÷1080＝1/12、5÷60＝1/12 で割合が一致 ✓）

  4. 小さくなりやすい。しるしのコイが集まっているすみの近くでつかまえると、標本の中のしるしの割合（4/50にあたる値）が池全体の割合より大きく出る。60/x＝(大きめの割合) を解くと x は小さくなるから、推定値は本当の総数より小さい方にかたよりやすい。 （確かめ: 本文の例で割合が0.08→0.16に倍増したとすると x＝60÷0.16＝375＜750 ✓ 割合が大きいほど x は小さい。）

stretch

S1 改善が期待できる点: 標本（2回目の捕獲）が大きくなるので、割合についても、平均で見たのと同じように標本が大きいほど値が安定する傾向が期待できる（この教材で実際に確かめたのは平均の例（L04）である）。推定が安定しやすくなると考えられる。 それでも言い切れない理由の例: 「ばらつきが小さくなるのは傾向であって、標本調査である以上、母集団についての確定的な判断は困難だから。」

L07 調査を批評する——結果を疑い、言い方を選ぶ

（記述問題は解答例。同じ観点が自分の言葉で書けていればよい。）

練習

  1. 例: 知りたい母集団は「みどり市の中学生」 なのに、質問した相手は「駅前の買い物中の大人」で、母集団にふくまれない人たちを調べている。標本が母集団から取り出されていないので、この結果から中学生の利用時間について語ることはできない。

  2. 解答例（3つ選んだ場合）:

    • 観点1（母集団の想定）: 来館者の回答を「市民の9割」と言いかえている → 結論を「回答した来館者のうち9割」に改め、市民全体を語りたいなら市民全体から標本をとる設計に直す。

    • 観点2（抽出方法）: 来館者への手渡しは、図書館をよく使う人にかたよる → 例: 市の住民名簿などから乱数で対象者を選んで送る形にする。

    • 観点3（回答率）: 1000枚中200枚（回答率20%）で、答えた人は関心の強い人にかたよるおそれ → 質問を短くする・答えやすい形式にするなど回収率を上げるくふうをし、回収率も結果と一緒に報告する。

    • 観点4（質問の誘導性）: 「便利な〜思いませんか」が同意をさそう → 選択肢を対等に並べた中立な質問文に直す。

    • 観点5（実施方法）: 職員に見える形の回答は本音を言いにくい → 無記名で投かんできる形にする。

  3. 例: 「朝読書を、わが校でも始めるべきだと思いませんか」→ 「朝読書の導入について、あなたの考えに近いものを選んでください。（導入すべきだ／どちらかといえば導入すべきだ／どちらかといえば導入しなくてよい／導入しなくてよい／分からない）」 （直す点: ①「授業もよく分かるようになると評判の」という根拠のない良い評判の刷りこみを削る ②「〜べきだと思いませんか」という同意をさそう形をやめ、賛否の選択肢を対等に並べる。）

  4. 例: 「この調査では、標本として選んだ生徒の平均読書時間は週3.5時間だった。このことから、みどり中の生徒全体の平均読書時間はおよそ週3.5時間と推定される。」（「〜であることが分かった」という確定の言い方を避け、標本で観察された事実と母集団への推定を区別して書けていればよい。）

  5. 解答例（形式が満たせていれば内容は自由）: (1) 母集団: 自分の学校の生徒全員（例: 480人）。 (2) 標本の大きさ: 例: 60人。理由の例: 「全員に聞くのは時間がかかりすぎるが、少なすぎるとばらつきが大きくなる傾向がある（L04）ので、手間とのつり合いでこの人数にした。」（人数そのものより、ばらつきと手間の両方に触れた理由づけができているかを見る。） (3) 手順の例: 「名簿の全員に001〜480の番号をつけ、コンピュータの乱数（または乱数さい）で3けたの数を作り、001≦番号≦480 の数だけを採用して60人選ぶ。範囲外・重複は読み飛ばす。」 (4) 質問文の例: 「平日1日あたりの家庭学習の時間に、いちばん近いものを選んでください。（30分未満／30分以上1時間未満／1時間以上2時間未満／2時間以上）」 (5) 報告文の例: 「標本60人の平均は◯時間だった。このことから、全校生徒の平均はおよそ◯時間と推定される。」 自己点検: 観点1〜5（母集団の想定・抽出方法・回答率・質問の誘導性・実施方法）それぞれについて1行ずつ確認できていればよい。

stretch

S1 例: 無作為抽出は「調べる側が、等しい確率になる仕組み（乱数など）で選ぶ」ことで成り立つ。回答する・しないを回答者自身が決める場合、選ばれ方が回答者の関心・性格・時間の余裕などに左右され、どの人も等しい確率で標本に入るとはいえなくなる。だから「回収できた分」は、たとえ数が多くても無作為抽出の標本とはみなせない。
