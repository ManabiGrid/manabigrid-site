L06 グラフを読む——増減は原点を境に

練習

  1. (1) 減る。(2) 増える。(3) y＝3×0²＝0。これはこの関数のyの値の中でもっとも小さい値（a＝3＞0で上に開くので、頂点の高さがいちばん低い）。

  2. y＝−2x²（a＜0・下に開く）:

    • x＜0の範囲では、xの値が増えるとyの値は増える。

    • x＝0のときy＝0で、これはyの値の中でもっとも大きい値。

    • x＞0の範囲では、xの値が増えるとyの値は減る。

    検算: 表 x＝−2, −1, 0, 1, 2 → y＝−8, −2, 0, −2, −8。左から読むと −8→−2→0（増）→−2→−8（減）。✓

  3. (ア) 誤。x＞0の範囲では増えるが、x＜0の範囲では減る（例: x＝−2→−1でy＝4→1）。 (イ) 正。x²≧0だから−x²≦0。yが0より大きくなることはない（もっとも大きい値はx＝0のときの0）。 (ウ) 正。一次関数の増減は全体で一方通行（a＝−3＜0なのでどこでも減る）。

  4. 解答例:「x＜0の範囲では、xの値が増えるとyの値は減る。x＞0の範囲では、xの値が増えるとyの値は増える。」（xの範囲＋増減がセットで2文になっていれば合格。「x＝0のときy＝0でもっとも小さい」を添えていればなおよい）

stretch

S1 xが−3から−1まで: yは9→1で、減少量は8。xが−1から0まで: yは1→0で、減少量は1。同じ「減る」でも、減り方の大きさは場所によってちがう（原点に近づくほど減り方がゆるやかになる）。 検算: (−3)²＝9、(−1)²＝1、9−1＝8。1−0＝1。✓

L07 変域——端の値だけでは決まらない

練習

  1. (1) 4≦y≦16。 検算: ①端の値: x＝2でy＝4、x＝4でy＝16。②変域2≦x≦4はx＝0を含まないので端の値だけでよい。③概形: x＞0では増えるだけの区間。✓ (2) 1≦y≦9。 検算: ①端の値: x＝−3でy＝9、x＝−1でy＝1。②x＝0を含まない。③概形: x＜0では減るだけの区間（yは9から1へ）。✓

  2. 0≦y≦8。 検算: ①端の値: x＝−2でy＝2×4＝8、x＝1でy＝2。②変域−2≦x≦1はx＝0を含む→a＝2＞0なのでy＝0がもっとも小さい値。③概形: 上に開く放物線の、頂点をまたぐ区間。もっとも大きい値は端の高い方の8。✓

  3. −9≦y≦0。 検算: ①端の値: x＝−1でy＝−1、x＝3でy＝−9。②変域−1≦x≦3はx＝0を含む→a＝−1＜0なのでy＝0がもっとも大きい値。③概形: 下に開く放物線の、頂点をまたぐ区間。もっとも小さい値は端の低い方の−9。✓

  4. 誤り: 変域−4≦x≦2はx＝0を含むのに、端の値だけで判断している点。x＝0のときy＝0が、この範囲でのyのもっとも小さい値になる（2ではない）。 正しい答え: 0≦y≦8。 検算: ①端の値8と2 ②x＝0を含む→y＝0 ③概形: 上に開く・頂点をまたぐ。もっとも大きい値は端の高い方の8。✓

stretch

S1 解答例（2つ以上あれば合格）: 0≦x≦3／−3≦x≦0／−3≦x≦3／−3≦x≦1　など。 共通の条件を整理すると:「−3以上3以下の範囲内で、x＝0を含み、−3と3の少なくとも一方を端に持つ」変域なら、すべて0≦y≦9になる。つまり答えは1通りに決まらない。 検算（例: −3≦x≦1）: ①端の値9と1 ②x＝0を含む→y＝0 ③もっとも大きい値は9——0≦y≦9。✓

L08 変化の割合は一定でない

練習

  1. (1) (9−1)÷(3−1)＝8÷2＝4 (2) (25−9)÷(5−3)＝16÷2＝8 (3) (1−9)÷(−1−(−3))＝(−8)÷2＝−4 検算: yの値の再計算 1²＝1, 3²＝9, 5²＝25, (−3)²＝9, (−1)²＝1。✓ (3)が負なのは、x＜0の範囲でyが減っている（L06）ことと整合。✓

  2. (1) y＝2x²: x＝1でy＝2、x＝4でy＝32。(32−2)÷(4−1)＝30÷3＝10 (2) y＝−x²: x＝1でy＝−1、x＝4でy＝−16。(−16−(−1))÷(4−1)＝(−15)÷3＝−5

  3. xが1から3: x＝1でy＝3、x＝3でy＝11。(11−3)÷2＝4。xが5から8: x＝5でy＝19、x＝8でy＝31。(31−19)÷3＝4。 コメント例:「一次関数は区間をどう選んでも変化の割合が4（傾きと同じ）で一定。y＝ax²との対比がはっきりした」。 検算: 4×1−1＝3、4×3−1＝11、4×5−1＝19、4×8−1＝31。✓

  4. 0秒後〜2秒後: 進んだ距離は2×2²−2×0²＝8−0＝8 (m)。平均の速さ＝8÷2＝4 (m/秒)。 2秒後〜4秒後: 2×4²−2×2²＝32−8＝24 (m)。平均の速さ＝24÷2＝12 (m/秒)。 読みの例:「同じ2秒間でも、あとの区間の平均の速さは3倍になっている。台車はだんだん速くなっている」。 検算: 2×16＝32、2×4＝8、(32−8)÷2＝12。✓

  5. 解答例:「変化の割合が一定であることは、グラフが直線になることと同じことだった。y＝x²では、xが1から3までの変化の割合は4、3から5までは8で、区間によって変わる。だから、y＝x²のグラフは直線にならず、曲線になる。」（①一定⇔直線の関係 ②計算値4と8の引用 ③「だから曲線」の結論、の3要素が入っていれば合格）

stretch

S1 x＝tでy＝t²、x＝t＋1でy＝(t＋1)²＝t²＋2t＋1。 変化の割合＝{(t²＋2t＋1)−t²}÷{(t＋1)−t}＝(2t＋1)÷1＝2t＋1。 t＝0, 1, 2, 3を代入すると1, 3, 5, 7——本文の差の行の奇数列と一致する。奇数列の正体は、区間幅1の変化の割合2t＋1（tが1増えるごとに2ずつ増える）だった。 検算: t＝2のとき2t＋1＝5。表でもx: 2→3でyは4→9、差5。✓

L09 四つの関数を見わたす

練習

  1. 完成した表:

    観点比例 y＝ax反比例 y＝a/x一次関数 y＝ax＋b（b≠0のとき）y＝ax²
    グラフの形原点を通る直線曲線（2本1組）直線原点を頂点とする放物線（曲線）
    変化の割合一定（aに等しい）一定でない一定（aに等しい）一定でない
    xがm倍になるとyもm倍yは1/m倍（きまった倍にならない）yはm²倍
    一定になっている商・積y÷xが一定x×yが一定（なし）y÷x²が一定

    検算（反比例の欄）: y＝12/xでx: 1→2の変化の割合は(6−12)÷1＝−6、x: 2→3では(4−6)÷1＝−2——たしかに一定でない（S1と同じ計算）。x×y＝12でどの列も一定。✓

  2. (ア) y÷x＝5, 5, 5, 5で一定 → 比例 y＝5x。（比例は一次関数の特別な場合（b＝0）なので、「一次関数にも当てはまる」と答えても誤りではない——いちばん細かい分類が比例、ということ） (イ) xが1増えるごとにyが2ずつ増える（変化の割合一定）が、x＝1でy＝3≠比例の形 → 一次関数 y＝2x＋1。検算: 2×4＋1＝9。✓ (ウ) y÷x²＝2, 2, 2, 2で一定 → y＝2x²。検算: 2×3²＝18。✓ (エ) x×y＝12, 12, 12, 12で一定 → 反比例 y＝12/x。検算: 12÷4＝3。✓

  3. 解答例:「xの値が2倍になったときのふるまいに注目すると（観点）、y＝2xではyも2倍になるが、y＝2x²ではyは4倍になる（比較）。実際、xが1から2になるとき、y＝2xではy＝2→4で2倍、y＝2x²ではy＝2→8で4倍である（根拠）。」 検算: 2×1＝2, 2×2＝4／2×1²＝2, 2×2²＝8。✓

  4. 解答例:「変化の割合に注目する。一次関数のグラフが直線なのは、変化の割合がどの区間でも一定だからだった。y＝x²の変化の割合は区間によって変わる（たとえば4や8）ので、グラフは直線にならず曲線になる。」（観点の宣言・一次関数との比較・数値または式の根拠、の3点がそろっていれば合格。3文以内）

stretch

S1 y＝12/x: xが1から2まで——(6−12)÷(2−1)＝−6。xが2から3まで——(4−6)÷(3−2)＝−2。一定でないので、反比例も「曲線組」だと確かめられる。 後半（自由記述）の考察例:「y＝x²は、xが大きくなるとyがいくらでも大きくなっていくが、y＝12/xは、xが大きくなるとyが0に近づいていく。増えていく先のふるまいがまったくちがうから、曲線の形（開いていく放物線と、軸に近づいていく曲線）もちがってくる」。（表の値の行き先に触れていれば合格）
