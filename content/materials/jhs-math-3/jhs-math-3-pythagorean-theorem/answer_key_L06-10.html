L06 √の長さを作図する——数直線上の平方根

練習1 1²＋3²＝1＋9＝10 だから、直角をはさむ2辺は1と3。 手順の解答例:

  1. 数直線をかき、0と3の点をとる。

  2. 3の点から数直線に垂直な線を上へ引く。

  3. 垂線上に、3の点から長さ1の点Pをとる（原点OからPまでが斜辺で、OP＝√(3²＋1²)＝√10）。

  4. コンパスの針をOに置き、半径OPの弧をかいて数直線と交わった点が√10。

検算: (√10)²＝10＝9＋1 ✓

練習2 2²＋3²＝4＋9＝13 → 2と3。検算: (√13)²＝13 ✓

練習3 原点から「横2・縦1」の直角三角形の斜辺を写しているから、x＝√(2²＋1²)＝√5。理由: 斜辺の長さが√5で、コンパスはその長さを変えずに数直線上へ移すから。検算: (√5)²＝5＝4＋1 ✓

練習4 解答例: 「√2の作図と同じ円（原点中心・半径√2）は、数直線の負の側とも交わる。その交点が−√2。」（0から左へ同じ長さをとる、という説明でもよい。）

S1 1²＋(√2)²＝1＋2＝3 だから、直角をはさむ2辺を1と√2にすればよい。 手順の例: ①まず本文の手順で√2の線分（1辺1の正方形の対角線）を作る。②数直線の1の点から垂線を立て、その上に長さ√2の点Qをコンパスでとる（①で作った√2を写す）。③原点OからQまでが斜辺で OQ＝√(1＋2)＝√3。④コンパスでOQを数直線に写す。 検算: (√3)²＝3 ✓

S2 作図の確認ポイント: 直前の斜辺（√n）を新しい直角三角形の1辺に使い、もう1辺は常に長さ1にすると、新しい斜辺は √(n＋1) になる（(√n)²＋1²＝n＋1）。√2→√3→√4（＝2になることも確認できる）→√5→√6…と続く。数直線への写し取りは各段階で可能。

L07 直方体の対角線——3段階法

練習1 ① 高さの辺は底面に垂直（直角はここ）。 ② 底面（6cm×8cm）を取り出す: 底面対角線²＝6²＋8²＝36＋64＝100 → 底面対角線＝10cm。次に「底面対角線10cm・高さ5cm」の直角三角形を取り出す。 ③ 対角線²＝10²＋5²＝100＋25＝125 → 対角線＝√125＝5√5 cm。 検算: (5√5)²＝125＝100＋25 ✓

練習2 底面対角線²＝6²＋6²＝72。対角線²＝72＋6²＝108 → 対角線＝√108＝6√3 cm。検算: (6√3)²＝108 ✓

練習3 底面対角線²＝2²＋3²＝13。対角線²＝13＋4²＝29 → √29 cm（29は素数）。検算: 4＋9＋16＝29 ✓

練習4 底面対角線²＝4²＋4²＝32。高さをhとすると h²＋32＝8²＝64 → h²＝32 → h＝√32＝4√2 cm。検算: 32＋32＝64 ✓

S1 底面対角線²＝a²＋b²。対角線²＝(a²＋b²)＋c² → 対角線＝√(a²＋b²＋c²)。 a＝b＝c＝4 の検算: √(16＋16＋16)＝√48＝4√3 → 例題2と一致 ✓

S2 1辺a、面の対角線√2a、空間の対角線√3a より、比は 1 : √2 : √3。L06のうずまき（√1→√2→√3）と同じ数の並びが、立方体の中にも現れている。

L08 錐体の高さと体積

練習1 ① 高さOHは底面に垂直（H＝底面の中心）。 ② 底面（1辺4cmの正方形）: 対角線＝4√2cm → 中心までの半分は2√2cm。取り出す直角三角形は「斜辺6cm・底辺2√2cm・高さOH」。 ③ OH²＝6²−(2√2)²＝36−8＝28 → OH＝√28＝2√7 cm。 検算: 28＋8＝36 ✓ 体積＝(1/3)×(4×4)×2√7＝32√7／3 cm³。

練習2 高さ²＝13²−5²＝169−25＝144 → 高さ＝12cm。検算: 144＋25＝169 ✓ 体積＝(1/3)×π×5²×12＝100π cm³。

練習3 母線²＝4²＋8²＝16＋64＝80 → 母線＝√80＝4√5 cm。検算: (4√5)²＝80 ✓

練習4 12√7≒12×2.6458＝31.7496 → 約31.7 cm³。（途中を√のまま運び、最後に1回だけ近似した。） 検算（最終桁の精度保証）: (12√7)²＝144×7＝1008。31.7²＝1004.89＜1008＜31.75²＝1008.0625 だから 31.7＜12√7＜31.75 で、小数第1位までの近似値は31.7で確定。与える近似値の桁は、答えの最終桁の精度を保証できる桁まで細かくとる——√7≒2.65（3けた）では、12倍で誤差が約0.05までふくらんで31.8となり、小数第1位がずれる。

S1 底面対角線の半分は (√2／2)a だから、その2乗は a²/2。 高さ²＝b²−a²/2 → 高さ＝√(b²−a²/2)。 成り立つ条件: 高さ²＞0 より b²＞a²/2（bとaは正だから b＞a/√2、整理すると √2·b＞a）。 等号（≧ではなく＞）である理由: b²＝a²/2 だと高さ0、つまり頂点が底面に落ちてぺしゃんこになり、立体（錐）にならないから。 検算: a＝6、b＝5（例題1）では 25＞18 ✓ で高さ√7と整合。

S2 HM＝底面の1辺の半分＝3cm（中心から辺の中点までの距離）。直角三角形OHMで OM²＝OH²＋HM²＝7＋9＝16 → OM＝4cm。 検算: (√7)²＋3²＝16＝4² ✓ 比較: OM＝4＞OH＝√7（√7≒2.6）——OMのほうが長い。斜めに立つ側面の高さは、まっすぐの高さより必ず長い（OHが直角をはさむ辺、OMが斜辺だから）。

L09 見渡せる距離——理想化と限界

練習1 d²＝(6400.5)²−6400²＝(6400.5＋6400)(6400.5−6400)＝12800.5×0.5＝6400.25 80²＝6400 だから √6400.25 は80のわずかに上 → 約80km。 検算: 6400.25−6400＝0.25 で、80.00…²との差はごく小さい ✓

練習2 d²＝(6400.3)²−6400²＝12800.3×0.3＝3840.09 61²＝3721、62²＝3844 だから √3840.09 は62の少し下 → 約62km。

練習3 解答例: 「実際には、途中の山や建物、大気の状態によって視界はさえぎられるし、地球も完全な球ではない。だからこの値は、さえぎるものが何もない場合の幾何学的な上限の目安である。」（「実際には」で始まり、単純化した条件に触れていれば正答。）

練習4 解答例: 「地球の複雑な形のままでは計算できないので、理想化——本質を保ったまま単純な図形（球）に置き換えること——をして、円の接線と直角三角形の問題に直すため。」

S1 d²＝(R＋h)²−R²＝R²＋2Rh＋h²−R²＝2Rh＋h²。 hがRに比べてきわめて小さいとき h²は2Rhに比べて無視できるほど小さいから、d²≒2Rh、つまり d≒√(2Rh)＝√(2R)×√h。√(2R) は定数だから、dは√hに比例する。 確認: R＝6400 で √(2Rh)＝√(12800h)。

  • h＝2: √25600＝160 ✓（例題1の160と一致）

  • h＝1: √12800≒113.1 ✓（例題2の約113と一致）

  • h＝0.5: √6400＝80 ✓（練習1の約80と一致）

S2 d∝√h なら、dを2倍にするには √h を2倍、つまり hを4倍にすればよい。（例で確認: h＝0.5→2（4倍）で d＝80→160（2倍）✓）

L10 章末まとめ

問1 (1) x²＝9²＋12²＝81＋144＝225 → 15cm。検算: 15²＝225 ✓ (2) x²＝7²−3²＝49−9＝40 → x＝√40＝2√10 cm。検算: 40＋9＝49 ✓

問2 最長辺をcにして判定。 (ア) 7²＋24²＝49＋576＝625＝25² → 直角 ✓ (イ) 4²＋6²＝16＋36＝52、7²＝49。52≠49 → 直角でない (ウ) (√2)²＋(√7)²＝2＋7＝9＝3² → 直角 ✓ 答え: (ア)と(ウ)

問3 高さ²＝8²−4²＝64−16＝48 → 高さ＝4√3cm。面積＝8×4√3÷2＝16√3 cm²。検算: (4√3)²＋16＝48＋16＝64 ✓

問4 弦の半分²＝7²−5²＝49−25＝24 → 半分＝2√6cm。弦＝4√6 cm。検算: 24＋25＝49 ✓

問5 横のずれ 3−(−1)＝4、縦のずれ −2−3＝−5 →長さ5。AB²＝16＋25＝41 → AB＝√41。検算: (√41)²＝41 ✓

問6 対角線²＝4²＋6²＋(2√3)²＝16＋36＋12＝64 → 8cm。検算: 8²＝64 ✓（L07のS1の式を使った。2段階で解いてもよい: 底面対角線²＝52、52＋12＝64。）

問7 高さ²＝10²−6²＝100−36＝64 → 高さ8cm。体積＝(1/3)×π×6²×8＝96π cm³。検算: 64＋36＝100 ✓

問8 解答例: 「直角三角形の3つの辺の上に、それぞれの辺を1辺とする正方形をかくと、面積はa²、b²、c²になる。三平方の定理 a²＋b²＝c² は、小さい2つの正方形の面積の和が、斜辺の上の正方形の面積とぴったり等しいことを表している。だから、この定理は長さの式であると同時に面積の関係でもある。」

S1 (1)

  • AB²＝6²＋2²＝40 → AB＝2√10

  • AC²＝2²＋4²＝20 → AC＝2√5

  • BC²＝(6−2)²＋(2−4)²＝16＋4＝20 → BC＝2√5

(2) 最長辺はAB。AC²＋BC²＝20＋20＝40＝AB² → 三平方の定理の逆により、∠Cを直角とする直角三角形といえる（AC＝BCなので直角二等辺三角形）。 (3) 直角をはさむ2辺はACとBC。面積＝(1/2)×2√5×2√5＝(1/2)×20＝10。 検算: 斜辺2√10＝(2√5)×√2 で直角二等辺の比1:1:√2と整合 ✓

S2 1辺aの正三角形の面積は (√3/4)a²。3つの辺の上の正三角形の面積は (√3/4)a²、(√3/4)b²、(√3/4)c²。 小さい2つの和＝(√3/4)a²＋(√3/4)b²＝(√3/4)(a²＋b²)。 三平方の定理より a²＋b²＝c² だから、これは (√3/4)c²、すなわち一番大きい正三角形の面積に等しい。成り立つ ✓ （同じ形の図形なら面積は「1辺の2乗×定数」なので、正方形でなくても定数がくくり出せて同じ関係が成り立つ、というのがからくり。）
