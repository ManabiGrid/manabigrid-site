L01 相似の意味と記号∽

例題1（本文解答の検算）

  • AB:EF=3:4.5 → 3×3=9、4.5×2=9 → =2:3 ✓

  • BC:FG=4:6 → 4×3=12、6×2=12 → =2:3 ✓

  • CD:GH=5:7.5 → 5×3=15、7.5×2=15 → =2:3 ✓

  • DA:HE=6:9 → 6×3=18、9×2=18 → =2:3 ✓

  • 4組すべて2:3で一致＋対応角がそれぞれ等しい（問題文で付与）→ 定義②により相似。判定「相似といえる」は正 ✓

例題2（本文解答の検算）

  • 対応: △ABC∽△PQR → A↔P、B↔Q、C↔R（並び順1・2・3文字目）

  • (1) B（2文字目）↔ Q（2文字目）→ Q ✓

  • (2) 辺CA=3文字目→1文字目 → 対応は3文字目→1文字目=R→P → 辺RP ✓

  • (3) R（3文字目）↔ C（3文字目）→ ∠C ✓

練習1

解答: 相似といえる。対応する辺の比（相似比）は1:2。

  • 5:10 → 5×2=10、10×1=10 → =1:2 ✓

  • 7:14 → 7×2=14 → =1:2 ✓

  • 9:18 → 9×2=18 → =1:2 ✓

  • 3組すべて1:2＋対応角等しい（問題文で付与）→ 相似 ✓

練習2

解答: 相似といえない。理由: 対応する辺の比がすべて等しくなっていないから（たての比は3:6=1:2だが、よこの比は5:9で1:2ではない）。

  • たて 3:6=1:2（3×2=6 ✓）

  • よこ 5:9 → 1:2なら5×2=10のはずだが10≠9 → 比が不一致 ✓

  • 長方形なので角はすべて90°で等しいが、辺の比の条件を満たさない → 非相似の判定は正 ✓

  • （誤概念チェック用: 「角が全部等しいから相似」への反例として機能することを確認）

L02 相似比と対応する辺・角

例題1（本文解答の検算）

  • (1) AB:DE=8:12 → 8×3=24、12×2=24 → 2:3 ✓

  • (2) 10:EF=2:3 → 交差積 10×3=30=2×EF → EF=15 ✓（別検算: 相似比2:3=×1.5、10×1.5=15 ✓）→ EF=15cm

  • (3) ∠A↔∠D（1文字目対応）→ ∠D=70° ✓（角は相似比によらず不変）

例題2（本文解答の検算）

  • AC（1・3文字目）↔PR（1・3文字目）: 9:PR=3:4 → 9×4=36=3×PR → PR=12 ✓ → PR=12cm

  • BC（2・3文字目）↔QR（2・3文字目）: BC:12=3:4 → 4×BC=36 → BC=9 ✓ → BC=9cm

  • 一貫性検算: AC:PR=9:12=3:4、BC:QR=9:12=3:4 → 全対応辺が同一相似比3:4 ✓

練習1

解答: DE=12cm、EF=18cm。

  • 相似比1:3。AB:DE=1:3 → 4:DE=1:3 → DE=4×3=12 ✓

  • BC:EF=1:3 → 6:EF=1:3 → EF=6×3=18 ✓

  • 一貫性: 4:12=1:3、6:18=1:3 ✓

練習2

解答: BC=8cm、∠B=65°。

  • 相似比: AB:DE=6:9 → 6×3=18、9×2=18 → =2:3 ✓

  • BC:EF=2:3 → BC:12=2:3 → 3×BC=24 → BC=8 ✓（別検算: 12×(2/3)=8 ✓）

  • ∠B↔∠E（2文字目対応）→ ∠B=∠E=65° ✓

練習3

解答: 周の長さは22.5cm。

  • △DEFの各辺: 4×(3/2)=6、5×(3/2)=7.5、6×(3/2)=9 ✓

  • 周: △ABC=4+5+6=15、△DEF=6+7.5+9=22.5 ✓

  • 周の比15:22.5 → 15×3=45、22.5×2=45 → =2:3=相似比 ✓（stretchの布石「周の比=相似比」成立を確認）

L03 三角形の相似条件

例題1（本文解答の検算）

  • (1) 4:6（4×3=12=6×2）=6:9（6×3=18=9×2）=8:12（8×3=24=12×2）=2:3 ✓ → 相似・対応する3組の辺の比がすべて等しい。△ABC∽△DEF ✓

  • (2) 5:7.5（5×3=15=7.5×2）=8:12（8×3=24=12×2）=2:3、間の角∠A=∠D=45° ✓ → 相似・対応する2組の辺の比とその間の角がそれぞれ等しい ✓

    • 「間の角」検証: ∠AはABとACの間、∠DはDEとDFの間。比を取った辺の組(AB,DE)(AC,DF)の間の角である ✓

  • (3) ∠C=180−50−70=60 ✓、∠F=180−50−60=70 ✓。角の組{50,60,70}が両者一致 ✓

    • 対応: A(50°)↔D(50°)、B(70°)↔F(70°)、C(60°)↔E(60°) → 記号は△ABC∽△DFE ✓（△ABC∽△DEFと書くと∠B=70°↔∠E=60°となり不成立——本文の注意書きどおり）

例題2（本文解答の検算）

  • AB:DE=6:8 → 6×4=24=8×3 → =3:4 ✓、BC:EF=9:12 → 9×4=36=12×3 → =3:4 ✓

  • 2組の辺の比は等しいが、その間の角（∠B・∠E）の情報なし → 条件2は適用不可。条件1（3組目のCA:FDが不明）・条件3（角の情報なし）も適用不可 → 「この情報だけでは相似といえない」は正 ✓

練習1

(ア) 解答: 相似といえる。条件: 対応する3組の辺の比がすべて等しい。△ABC∽△DEF（相似比2:3）。

  • 3:4.5 → 3×3=9=4.5×2 → 2:3 ✓

  • 5:7.5 → 5×3=15=7.5×2 → 2:3 ✓

  • 7:10.5 → 7×3=21=10.5×2 → 2:3 ✓

(イ) 解答: 相似といえない。理由: △ABCの角の組は{55°,60°,65°}、△DEFの角の組は{55°,70°,55°}で一致しないから（対応する2組の角がそろわない）。

  • ∠C=180−55−60=65 ✓、∠F=180−55−70=55 ✓

  • 集合比較: {55,60,65}≠{55,55,70} ✓ → 非相似の判定は正 ✓

(ウ) 解答: 相似といえる。条件: 対応する2組の辺の比とその間の角がそれぞれ等しい。△ABC∽△DEF（相似比2:5）。

  • AB:DE=4:10 → 4×5=20=10×2 → 2:5 ✓

  • AC:DF=6:15 → 6×5=30=15×2 → 2:5 ✓

  • 間の角: ∠A=∠D=50°。∠AはAB・ACの間、∠DはDE・DFの間で、比を取った辺の組の間の角 ✓

練習2

解答: 合同である。相似比1:1なら対応する辺の比がすべて1:1、つまり対応する3組の辺がそれぞれ等しくなり、合同条件「3組の辺がそれぞれ等しい」を満たすから。

  • 論理検算: 相似比1:1 ⇒ 対応辺の長さが等しい ⇒ 3辺相等 ⇒ 合同 ✓（相似は合同を含む仲間分け——L01定義①の「拡大率1倍なら合同」とも整合 ✓）

検算総括

  • 比の検算: 本書に記載した比の照合はすべて交差積で実施し、不一致なし（非相似判定用の意図的不一致2件——L01練習2の5:9、L03練習1(イ)の角の組——は「不一致であること」を確認済み）。

  • 内角和検算: 角度を3つ扱う設問（L03例題1(3)・練習1(イ)）はすべて180°との整合を確認。

  • 対応の検算: 記号の並び順と辺・角の対応をすべて文字位置で照合。L03例題1(3)は対応順が入れ替わる設計（△ABC∽△DFE）であることを再確認——本文・解答とも一致。

  • 数値の出所: 全例題・練習の図形と数値は本ドラフトの自作。教科書・過去問の数値設定の転載なし。
