自己採点用（生徒も見てよい。解いてから開こう）。

すべての例題・練習を再計算した。検算は交差積・内角和180°・対応順（文字位置照合）の3点で行った。

L04 相似の証明①（構想）

例題1（本文解答の検算）

  • 対応: △OAB∽△ODC → O↔O、A↔D、B↔C（文字位置1・2・3）

  • OA:OD=4:6 → 4×3=12、6×2=12 → =2:3 ✓

  • OB:OC=6:9 → 6×3=18、9×2=18 → =2:3 ✓

  • 間の角: ∠AOBはOA・OBの間、∠DOCはOD・OCの間＝比を取った辺の組の間の角 ✓。点Oで線分AD・BCが交わるので∠AOBと∠DOCは対頂角 ✓

  • 方針メモの条件選択「対応する2組の辺の比とその間の角」は正 ✓

例題2（本文解答の検算）

  • 対応: △ADE∽△ACB → A↔A、D↔C、E↔B

  • ∠AED（△ADEの3文字目Eの角）↔∠ABC（△ACBの3文字目Bの角）＝仮定の等角と対応一致 ✓

  • ∠A（1文字目）は共通 ✓ → 2組の角がそろう。残り1組∠ADE↔∠ACBは内角和により自動的に等しい（180°−∠A−∠AED=180°−∠A−∠ABC）✓

  • 辺の長さの与件ゼロ → 比の条件は使用不能、「対応する2組の角」一択の記述は正 ✓

練習1

解答: △PAC∽△PBD（相似比4:3）。

△PAC∽△PBD を示したい ← 対応する2組の辺の比とその間の角 ← PA:PB=PC:PD=4:3〔仮定の長さ〕、∠APC=∠BPD〔対頂角は等しい〕

  • PA:PB=8:6 → 8×3=24、6×4=24 → =4:3 ✓

  • PC:PD=12:9 → 12×3=36、9×4=36 → =4:3 ✓

  • ∠APC・∠BPD: A・Bは線分AB上でPの両側、C・Dは線分CD上でPの両側 → 対頂角 ✓

  • 対応: P↔P、A↔B、C↔D。比を取った辺(PA,PB)(PC,PD)と記号順が一致 ✓

  • もう1つの組み方の棄却検算: △PADと△PBCなら PA:PB=8:6=4:3 だが PD:PC=9:12=3:4（9×4=36、12×3=36で3:4 ✓）→ 4:3≠3:4（4×4=16≠3×3=9）→ この組は比がそろわず、この情報からは相似と結論できない ✓（相似になる組の特定が一意であることを確認）

練習2

解答: △BAD∽△BCA。

△BAD∽△BCA を示したい ← 対応する2組の角がそれぞれ等しい ← ∠BAD=∠BCA〔仮定〕、∠Bは共通〔共通な角〕

  • 対応: B↔B、A↔C、D↔A（文字位置照合）。∠BAD（2文字目Aの角）↔∠BCA（2文字目Cの角）＝仮定の等角 ✓、∠B（1文字目）共通 ✓

  • Dは辺BC上にあるので∠ABD=∠ABC（同じ角）＝共通の扱いは正 ✓

L05 相似の証明②（記述と検査）

例題1（本文証明の検算）

  • 対応: △DBA∽△ABC → D↔A、B↔B、A↔C

  • ∠BDA（Dの角）↔∠BAC（Aの角）: ともに90°〔仮定: 垂線・直角〕✓

  • ∠B共通（B↔B）✓

  • 第3の角の内角和検算: △DBAで∠DAB=180°−90°−∠B=90°−∠B。△ABCで∠C=180°−90°−∠B=90°−∠B → ∠DAB=∠C（対応A↔Cと一致）✓

  • 3点検の実演: ①②＝仮定、④＝共通な角、結論を使った根拠なし → 「合格」判定は正 ✓

例題2（本文解答の検算）

  • 誤り箇所の特定: ①「△OAB∽△ODCだから」＝結論そのものを根拠に使用＝循環論法。3点検の点検3で検出される設計 ✓

  • ②対頂角のみでは角1組 → 「対応する2組の角」は適用不可（角の相等はもう調達できない）→ 辺の比への切り替えは必然 ✓

  • 直した証明の数値（L04例題1と同一の図・同一数値）: OA:OD=4:6=2:3（4×3=12=6×2 ✓）、OB:OC=6:9=2:3（6×3=18=9×2 ✓）、∠AOB=∠DOC対頂角 ✓ → 「対応する2組の辺の比とその間の角」成立 ✓

練習1

解答（証明例）:

△BADと△BCAで、 ∠BAD=∠BCA …①〔仮定〕 ∠ABD=∠CBA（共通） …②〔共通な角〕 ①、②より、対応する2組の角がそれぞれ等しいので △BAD∽△BCA

3点検: 結論＝△BAD∽△BCAに印 → ①は仮定・②は共通な角 → 根拠に結論（またはそこから導かれる辺の比・∠BDA=∠BACなどの相等）を使っていない → 合格。

  • 対応検算: B↔B、A↔C、D↔A。①の等角＝2文字目どうし ✓、②＝1文字目どうし ✓（L04練習2の方針メモと一致）

練習2

(1) 解答（証明例）:

△OABと△ODCで、 OA:OD=6:9=2:3 …①〔仮定〕 OB:OC=4:6=2:3 …②〔仮定〕 ①、②より OA:OD=OB:OC …③ ∠AOB=∠DOC …④〔対頂角は等しい〕 ③、④より、対応する2組の辺の比とその間の角がそれぞれ等しいので △OAB∽△ODC

  • OA:OD=6:9 → 6×3=18、9×2=18 → =2:3 ✓

  • OB:OC=4:6 → 4×3=12、6×2=12 → =2:3 ✓

  • 3点検: ①②仮定・④対頂角、循環なし → 合格 ✓

(2) 解答: DC=7.5cm。

  • 対応: △OAB∽△ODC → 辺AB（2・3文字目）↔ 辺DC（2・3文字目）✓

  • 相似比2:3より AB:DC=2:3 → 5:DC=2:3 → 交差積 5×3=15=2×DC → DC=7.5 ✓（別検算: 5×1.5=7.5 ✓）

  • 根拠の筋: 相似を(1)で示したあとに「相似な図形では対応する線分の比はすべて等しい」〔相似な図形の性質〕を使う——L04主概念1の「性質は相似が示せたあとの根拠」と整合 ✓

stretch（検算のみ・解答は本文ヒントに準拠）

  • △DBA∽△DAC: 対応D↔D、B↔A、A↔C。∠BDA=∠ADC=90°（ADは垂線・BCは直線）✓。∠DBA=90°−∠C（△ABC内角和）、∠DAC=90°−∠C（△DAC内角和）→ 等しい ✓

  • AD求値: DB:DA=DA:DC → DA²=DB×DC=4×9=36 → AD=6cm。検算: 4:6 → 4×3=12=6×2 → 2:3、6:9 → 6×3=18=9×2 → 2:3 → 両比一致 ✓（交差積 4×9=36=6×6 ✓）。平方根の学習範囲: 36は6の平方なので√記号不要（中3・平方根既習でもあるため問題なし）✓

検算総括

  • 比の検算: 記載した比の照合はすべて交差積で実施し、不一致なし（意図的不一致1件——L04練習1の棄却検算4:3≠3:4——は「不一致であること」を確認済み）。

  • 内角和検算: 第3の角の相等を使う場面（L04例題2・L05例題1・L05 stretch）はすべて180°との整合を確認。

  • 対応の検算: 相似の記号の並び順と辺・角の対応をすべて文字位置で照合。証明中の等角・辺の比と記号順の不整合なし。

  • 循環論法の教材内検査: 例題・練習の「正しい証明」6本（L05例題1・例題2修正版・練習1・練習2(1)＋L04方針メモ2本の根拠構成）について、根拠に結論由来の事実が混入していないことを個別確認。意図的な循環1件（L05例題2の誤答例）は循環「であること」を確認済み。

  • 数値の出所: 全例題・練習の図形と数値は本ドラフトの自作。教科書・過去問の数値設定の転載なし（垂線型の図の「型」のみ解説p.144の指導例に拠るが、数値・記号配置は自作）。
