L01 単項式と多項式の乗法・除法

準備運動（前提診断）

  1. 3(x＋2y)＝3x＋6y

  2. 5a＋2b−3a＋4b＝2a＋6b

  3. (−2)×(−3)＝6、(−2)×3＝−6（マイナス×マイナスはプラス、マイナス×プラスはマイナス）

  4. 30＝2×3×5

  5. 素数は 2と7。（1は素数に含めない。6＝2×3、9＝3×3 は素数ではない）

  6. 例: 12＝2×6（1×12、2×2×3 なども正解）

練習

  1. (1) 4x(2x＋3)＝4x×2x＋4x×3＝8x²＋12x (2) −2a(a−4b)＝(−2a)×a＋(−2a)×(−4b)＝−2a²＋8ab (3) (3x−6y)×(−5x)＝3x×(−5x)＋(−6y)×(−5x)＝−15x²＋30xy

  2. (1) (9a²＋6ab)÷3a＝9a²/(3a)＋6ab/(3a)＝3a＋2b 検算: 3a(3a＋2b)＝9a²＋6ab ✓ (2) (8x²−12xy)÷(−4x)＝8x²/(−4x)−12xy/(−4x)＝−2x＋3y 検算: (−4x)(−2x＋3y)＝8x²−12xy ✓

  3. 横＝面積÷縦＝3a(2a＋7b)÷3a＝2a＋7b

  4. まちがい: かっこの中の5に2xをかけ忘れている（後ろの項にもかける）。正しくは 2x(x＋5)＝2x²＋10x。 検算（項数の照合）: かっこの中が2項なので、答えも2項になるはず。2x²＋5 も2項だが、x＝1 を入れると左辺 2×6＝12、誤答 2＋5＝7 で不一致、正答 2＋10＝12 で一致 ✓

stretch

S1 a＝1, b＝2, c＝6 で確かめる。 (a＋b)÷c＝3÷6＝1/2。a/c＋b/c＝1/6＋2/6＝1/2 ✓ 一致する。 c÷(a＋b)＝6÷3＝2。c/a＋c/b＝6＋3＝9。2≠9 ✗ 一致しない。 説明: 分けてよいのは割られる数（分子側）が和のとき。ma＋mb＝m(a＋b) の形（分配法則）は分子の和にしか対応しておらず、割る数（分母側）の和は分けられない。（どちらの形でも、割る数が0でないことが前提——文字で割るときは、その文字が0でない場合を考えている。）

L02 式の展開——多項式×多項式

練習

  1. (1) (a＋2)(b＋3)＝ab＋3a＋2b＋6（予告どおり2×2＝4項・同類項なし） (2) (x＋4)(x＋6)＝x²＋6x＋4x＋24＝x²＋10x＋24 (3) (x−5)(x＋2)＝x²＋2x−5x−10＝x²−3x−10 (4) (3a＋b)(2a−5b)＝6a²−15ab＋2ab−5b²＝6a²−13ab−5b²

  2. (1) 予告: 3×2＝6項。(a＋b＋1)(c＋2)＝ac＋2a＋bc＋2b＋c＋2（同類項なし・6項 ✓）

  3. 図で、縦を x と 3、横を x と 5 に区切ると4つの部屋（x²・5x・3x・15）ができる。8x は、横長の部屋 5x と縦長の部屋 3x の2部屋の合計（5x＋3x）から来る。

  4. まちがい: 4×(−7)＝−28 なのに ＋28 としている（定数項の符号）。正しくは (x＋4)(x−7)＝x²−7x＋4x−28＝x²−3x−28。 検算: x＝1 で左辺 5×(−6)＝−30、x²−3x−28 に x＝1 で 1−3−28＝−30 ✓

stretch

S1 (a＋b)(c＋d＋e): 予告 2×3＝6項。展開すると ac＋ad＋ae＋bc＋bd＋be（6項 ✓）。 (a＋b＋c)(d＋e＋f): 3×3＝9項（ad＋ae＋af＋bd＋be＋bf＋cd＋ce＋cf）。 理由: 面積の図で、縦を左の項数だけ、横を右の項数だけ区切ると、部屋の数は（縦の区切り数）×（横の区切り数）になるから。左の各項と右の各項の組み合わせが1部屋に1つずつ対応する。

S2 まず (x＋1)(x＋2)＝x²＋3x＋2。これに (x＋3) をかける。 (x²＋3x＋2)(x＋3)＝x³＋3x²＋3x²＋9x＋2x＋6＝x³＋6x²＋11x＋6 順番を変えて (x＋2)(x＋3)＝x²＋5x＋6、これに (x＋1): x³＋x²＋5x²＋5x＋6x＋6＝x³＋6x²＋11x＋6 ✓ 同じ結果。

L03 乗法の公式①

練習

  1. (1) 和11・積30 → x²＋11x＋30 (2) 和 8＋(−3)＝5・積 8×(−3)＝−24 → x²＋5x−24 (3) 和 −7＋4＝−3・積 −28 → x²−3x−28 (4) 和 −5＋(−9)＝−14・積 45 → x²−14x＋45

  2. (1) 2×□＝14 より □＝7。確かめ: 和 2＋7＝9 ✓ (2) 左の□: (−3)×□＝−24 より □＝8。右の□: 和 −3＋8＝5。 検算: (x−3)(x＋8)＝x²＋5x−24 ✓

  3. かっこの中を整理: y＋6−1＝y＋5。よって (y−6)(y＋5)＝和 −1・積 −30 → y²−y−30

  4. 分配法則で開く: (x＋a)(x＋b)＝x²＋bx＋ax＋ab。同類項 bx＋ax をまとめて (a＋b)x。 よって (x＋a)(x＋b)＝x²＋(a＋b)x＋ab が再現できる。

stretch

S1 和10・積21 → 3と7。和10・積25 → 5と5（同じ数）。和10・積24 → 4と6。 （和を固定して積を大きくすると2数が近づく。積25＝最大は5と5のとき——L06の雑談も参照）

L04 乗法の公式②③④

練習

  1. (1) (x＋6)²＝x²＋12x＋36 (2) (x−9)²＝x²−18x＋81（最後の項は (−9)²＝＋81） (3) (a＋5)(a−5)＝a²−25 (4) (2x＋3)²＝(2x)²＋2×2x×3＋3²＝4x²＋12x＋9 (5) (3a−4b)(3a＋4b)＝(3a)²−(4b)²＝9a²−16b²

  2. (1) 別々の数 → 公式①: 和 −13・積 36 → x²−13x＋36 (2) 符号だけ違う → 公式④: x²−100 (3) 同じかっこの2乗 → 公式③: x²−2x＋1 (4) 別々の数 → 公式①: 和 6・積 −16 → a²＋6a−16

  3. (1) x＋y＝M と置く。(M＋2)(M−2)＝M²−4＝(x＋y)²−4＝x²＋2xy＋y²−4 (2) a−b＝M と置く。(M＋5)²＝M²＋10M＋25＝(a−b)²＋10(a−b)＋25＝a²−2ab＋b²＋10a−10b＋25

  4. a＝3, b＝4 のとき、(3＋4)²＝7²＝49。一方 3²＋4²＝9＋16＝25。49≠25 なので (a＋b)²＝a²＋b² は正しくない。 正しい展開 a²＋2ab＋b² との差は 2ab の項（この例では 2×3×4＝24。実際 49−25＝24 ✓）。

stretch

S1 差: (a＋b)²−(a−b)²＝(a²＋2ab＋b²)−(a²−2ab＋b²)＝4ab 和: (a＋b)²＋(a−b)²＝2a²＋2b² 検算（a＝5, b＝2）: 差 7²−3²＝49−9＝40＝4×5×2 ✓、和 49＋9＝58＝2×25＋2×4 ✓

S2 は設問ではなく、解き方の道筋（公式④→x²をひとまとまりとみてもう一度公式④）を示した「発展の入り口」のため、解答はなし（この先の4乗の式の展開・因数分解は高校範囲として本文で予告のみ）。
