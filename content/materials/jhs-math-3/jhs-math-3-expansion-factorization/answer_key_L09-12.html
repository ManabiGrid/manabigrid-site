  2. (1) 8×27＋8×73＝8×(27＋73)＝8×100＝800（共通因数8のくくり出し） (2) 15×99＋15＝15×99＋15×1＝15×(99＋1)＝15×100＝1500

  4. 49と51は「50−1」と「50＋1」。公式④（和と差の積） で、 49×51＝(50−1)(50＋1)＝50²−1²＝2500−1＝ 2499

S1 一の位が5の2けたの数を 10n＋5（nは十の位の数）と表す。 (10n＋5)²＝100n²＋2×10n×5＋25＝100n²＋100n＋25＝100n(n＋1)＋25 100n(n＋1) は「n(n＋1) を100倍した数」だから、下2けたは必ず25、その上の位は n(n＋1)＝十の位の数×（十の位の数＋1）。 85²: n＝8 → 8×9＝72 → 7225　検算: 85²＝(80＋5)²＝6400＋800＋25＝7225 ✓

L10 式による説明①

練習

  1. nを整数とすると、連続する2つの奇数は 2n＋1, 2n＋3、間の偶数は 2n＋2 と表される。 (2n＋1)(2n＋3)＋1＝4n²＋6n＋2n＋3＋1＝4n²＋8n＋4＝4(n²＋2n＋1)＝4(n＋1)²＝(2(n＋1))²＝ (2n＋2)² よって、連続する2つの奇数の積に1をたすと、間の偶数の2乗になる。 検算（例）: 3×5＋1＝16＝4² ✓、7×9＋1＝64＝8² ✓

  2. nを整数とし、連続する3つの整数を n−1, n, n＋1 と表す。 (n−1)＋n＋(n＋1)＝3n 3n は真ん中の数 n の3倍だから、連続する3つの整数の和は真ん中の数の3倍になる。 検算（例）: 5＋6＋7＝18＝3×6 ✓

  3. ゴールの形: 「2つの整数の和」＝ n＋(n＋1)。nを整数とし、続いた2つの整数を n, n＋1 と表す。 (n＋1)²−n²＝n²＋2n＋1−n²＝2n＋1＝n＋(n＋1) よって、2つの続いた整数の2乗の差は、その2つの整数の和に等しい。 検算（例）: 6²−5²＝11＝5＋6 ✓ （別解: 和と差の積で (n＋1)²−n²＝(n＋1＋n)(n＋1−n)＝(2n＋1)×1——因数分解でゴールの形が直接現れる）

  4. 不備: 偶数を 2n、奇数を 2n＋1 と同じ文字で表すと、「6と7」のようにとなり合う偶数・奇数の組しか表せない。「4と11」のような離れた組を含むすべての場合を扱うには、別々の文字を使う。 直し: mとnを整数とし、偶数を 2m、奇数を 2n＋1 と表す。和は 2m＋(2n＋1)＝2(m＋n)＋1。m＋n は整数だから、2(m＋n)＋1 は奇数。よって偶数と奇数の和は奇数である。

stretch

S1 n(n＋8)−(n＋1)(n＋7)＝(n²＋8n)−(n²＋7n＋n＋7)＝(n²＋8n)−(n²＋8n＋7)＝−7 結果に n が残らないので、田の字を並びのどこで取っても、対角線どうしの積の差は常に −7（左上×右下のほうが7小さい）で変わらない。 検算（例）: n＝1 → 1×9−2×8＝9−16＝−7 ✓

L11 式による説明②

練習

  1. もとの正方形の面積は x²。長方形の面積は (x＋2)(x−2)＝x²−4。 x²−4 は x² より4小さい。よって長方形の面積はもとの正方形より 4cm²小さくなる。

  2. のばす・縮める長さを a とすると、長方形の面積は (x＋a)(x−a)＝x²−a²——もとの正方形より a² 小さい。 式を読む: 差 a² には x が入っていない。つまり「同じ量だけのばして縮めると、面積は必ず小さくなり、減る量は（正方形の大きさによらず）のばした長さの2乗になる」。

  3. ステップ①: 道の外側は縦 p＋2a・横 q＋2a の長方形。 S＝(p＋2a)(q＋2a)−pq＝pq＋2ap＋2aq＋4a²−pq＝2ap＋2aq＋4a²＝2a(p＋q＋2a) ℓ＝真ん中の一周＝縦 (p＋a)・横 (q＋a) の長方形の周＝2{(p＋a)＋(q＋a)}＝2(p＋q＋2a) よって aℓ＝a×2(p＋q＋2a)＝2a(p＋q＋2a)＝S。S＝aℓ は長方形でも成り立つ ✓

  4. S＝π(r＋a)²−πr²＝π(r²＋2ar＋a²)−πr²＝π(2ar＋a²)＝πa(2r＋a) ℓ＝2π(r＋a/2)＝π(2r＋a) aℓ＝a×π(2r＋a)＝πa(2r＋a)＝S。円でも S＝aℓ が成り立つ ✓

stretch

S1 (x＋a)(x−b)＝x²＋(a−b)x−ab。これが x² に等しいためには、 (a−b)x−ab＝0　すなわち　(a−b)x＝ab 読み取れること（例）:

  • a＝b（同じ量ののばし縮め）だと左辺が0になり、ab＞0 と矛盾——練習2のとおり、同じ量では絶対に同じ面積にならない。

  • 同じ面積になるには a＞b（のばすほうを長く）が必要で、そのとき x＝ab/(a−b)。x が式に残っている＝この条件は正方形の大きさに関係する。たとえば a＝2, b＝1 なら x＝2 の正方形のときだけ面積が等しくなる（確かめ: (2＋2)(2−1)＝4＝2² ✓）。

L12 章末まとめ

A 展開

  1. (x＋7)(x−7)＝x²−49（公式④）

  2. (a−9)²＝a²−18a＋81（公式③）

  3. (x＋3)(x−8)＝和−5・積−24 → x²−5x−24（公式①）

  4. (2x＋5y)²＝(2x)²＋2×2x×5y＋(5y)²＝4x²＋20xy＋25y²（公式②）

  5. x＋y＝M と置く。(M−4)(M＋4)＝M²−16＝(x＋y)²−16＝x²＋2xy＋y²−16（置き換え＋公式④①）

B 因数分解

  1. 積56・和−15 → (−7,−8) → (x−7)(x−8)　検算: x²−15x＋56 ✓

  2. 共通因数3 → 3(x²−16) → 3(x＋4)(x−4)　検算: 3(x²−16)＝3x²−48 ✓

  3. 3点点検: 121＝11²、2×x×11＝22x ✓ → (x＋11)²　検算: x²＋22x＋121 ✓

  4. 共通因数2 → 2(a²−2ab＋b²) → 2(a−b)²　検算: 2(a²−2ab＋b²)＝2a²−4ab＋2b² ✓

  5. x−1＝M と置く。M²−9M＋20＝積20・和−9 → (−4,−5) → (M−4)(M−5)＝(x−1−4)(x−1−5)＝ (x−5)(x−6) 検算: (x−5)(x−6)＝x²−11x＋30。もとの式を展開して確認: (x−1)²−9(x−1)＋20＝x²−2x＋1−9x＋9＋20＝x²−11x＋30 ✓ 一致

C 利用

  1. 105×95＝(100＋5)(100−5)＝10000−25＝9975 53²−47²＝(53＋47)(53−47)＝100×6＝600

  2. nを整数とし、連続する2つの整数を n, n＋1（大きいほうは n＋1）と表す。 n(n＋1)＋(n＋1)＝(n＋1)(n＋1)＝ (n＋1)² ……共通因数 (n＋1) のくくり出し よって、連続する2つの整数の積に大きいほうの整数をたすと、大きいほうの整数の2乗になる。 検算（例）: 4×5＋5＝25＝5² ✓

  3. 残りの面積＝x²−a²×4＝x²−4a²（展開した形）＝(x)²−(2a)²＝ (x＋2a)(x−2a)（因数分解した形） 検算: (x＋2a)(x−2a)＝x²−4a² ✓ 読み取りの例: 展開した形からは「切り取った合計が 4a² で、正方形1枚あたり a²」が読める。因数分解した形からは「残りの面積は、縦 (x−2a)・横 (x＋2a) の長方形と同じ面積」——切って並べ替えられそうだ、と読める（L08 S2と同じ構図）。
