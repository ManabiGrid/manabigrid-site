L08 解を吟味する——もとの事象に戻す

練習

  1. ① もとの数をx（正の数）とおく。② 着目: 2乗した結果。x² と 2x＋35。③ x²＝2x＋35 → x²−2x−35＝0 → (x−7)(x＋5)＝0 → x＝7、x＝−5。 ④ xは正の数だから x＝−5 は不適。x＝7 を戻して: 7²＝49、2×7＋35＝49 ✓。答え 7

  2. ① 小さい方の整数をxとおく。② 着目: 2数の積。x(x＋1) と 110。③ x(x＋1)＝110 → x²＋x−110＝0 → (x＋11)(x−10)＝0 → x＝10、x＝−11。 ④ 問題文は「整数」なので負も可。x＝10 のとき 10と11（10×11＝110 ✓）。x＝−11 のとき −11と−10（(−11)×(−10)＝110 ✓）。答え 「10と11」と「−11と−10」の2組

  3. ① 横をx cm（x＞5。縦x−5が正であるため）とおくと、縦は (x−5)cm。② 着目: 面積。x(x−5) と 84。③ x(x−5)＝84 → x²−5x−84＝0 → 和が−5・積が−84の組: −12と7 → (x−12)(x＋7)＝0 → x＝12、x＝−7。 ④ x＞5 より x＝−7 は不適。x＝12 を戻して: 縦 12−5＝7cm、面積 7×12＝84cm² ✓。答え 縦7cm・横12cm

stretch

S1 「ある数」なら、④で除いた x＝−5 が復活する。確かめ: (−5)²＝25、2×(−5)＋35＝−10＋35＝25 ✓——たしかに問題の条件を満たす。答えは 7 と −5 の2つになる。方程式も解も変わらないのに、問題文の一語（「正の」）だけで答えが変わる——吟味が「問題文と照らし合わせる作業」であることの実例。

補足: 例3（√を含む解の吟味・本文の例）の検算

  • 厳密解: x(x−4)＝6 → x²−4x−6＝0 → (x−2)²＝10 → x＝2±√10。代入検算: x＝2±√10 のとき x²＝14±4√10、−4x＝−8∓4√10 なので x²−4x−6＝(14−8−6)＋(4√10−4√10)＝0 ✓（複号のどちらでも成立）。

  • 見積もり（2乗の挟み撃ち）: 3²＝9＜10＜16＝4² より 3＜√10＜4。さらに 3.1²＝9.61＜10＜10.24＝3.2² より 3.1＜√10＜3.2。よって 2−√10 は −1.2＜2−√10＜−1.1 の負の数で x＞4 に不適、2＋√10 は 5.1＜2＋√10＜5.2 で x＞4 に適する（吟味の結論自体は 3＜√10＜4 の桁で確定）。

  • 丸めの確かめ（「約5.2」の根拠）: 5.1＜2＋√10＜5.2 だけでは、小数第1位に丸めた値が5.1か5.2か決まらない（5.15より上か下かが分からないため）。そこでもう一歩: 3.15²＝9.9225＜10 だから √10＞3.15、よって 2＋√10＞5.15。あわせて 2＋√10＜5.2 なので 5.15＜2＋√10＜5.2 となり、小数第1位に丸めると 5.2 で確定する。

  • もとの事象での検算: 横＝(2＋√10)−4＝√10−2（3＜√10 より正）。面積＝(√10＋2)(√10−2)＝10−4＝6m² ✓（近似によらず厳密に一致）。

L09 利用の総仕上げ——図形と数の問題

練習

  1. ① 高さをx cm（x＞0）とおくと、底辺は (x＋4)cm。② 着目: 面積。x(x＋4)/2 と 48。③ x(x＋4)/2＝48 → 両辺を2倍して x(x＋4)＝96 → x²＋4x−96＝0 → (x＋12)(x−8)＝0 → x＝8、x＝−12。 解き方の理由の例: 「＝0に整理したら、和が4・積が−96の組（12と−8）が見つかったので因数分解で」。 ④ x＞0 より x＝−12 は不適。x＝8 を戻して: 底辺12cm、面積 8×12÷2＝48cm² ✓。答え 高さ8cm・底辺12cm

  2. ① もとの1辺をx cm（x＞6。4すみから3cmずつ切り取るので底面の1辺 x−6 が正であるため）。底面は1辺(x−6)cmの正方形、深さ3cm。② 着目: 容積。3(x−6)² と 108。③ 3(x−6)²＝108。 解き方の理由の例: 「(かたまり)²＝数 の形になるので、展開せず平方根の考えで」。 両辺を3で割って (x−6)²＝36 → x−6＝±6 → x＝12、x＝0。 ④ x＞6 より x＝0 は不適（そもそも紙がない）。x＝12 を戻して: 底面は1辺 12−6＝6cm、容積 6×6×3＝108cm³ ✓。答え 12cm

  3. ① その整数をxとおく（xは正の整数）。② 着目: 2つの計算結果の差。x²−2x と 63。③ x²−2x＝63 → x²−2x−63＝0 → (x−9)(x＋7)＝0 → x＝9、x＝−7。 ④ xは正の整数だから x＝−7 は不適。x＝9 を戻して: 9²＝81、9×2＝18、81−18＝63 ✓。答え 9

stretch

S1 底辺をx cm（x＞4。高さ x−4 が正であるため）とおくと、高さは (x−4)cm。 x(x−4)/2＝48 → x(x−4)＝96 → x²−4x−96＝0 → (x−12)(x＋8)＝0 → x＝12、x＝−8。 x＞4 より x＝12。底辺12cm・高さ8cm——練習1と同じ三角形に着く ✓（式は x²＋4x−96＝0 から x²−4x−96＝0 に変わったのに、答えの三角形は同じ。xのおき方は答えを変えない）。

L10 章末まとめ——「数と式」三年間の到達点

A. 解く（選ぶ理由は一例）

  1. x²＝64 → 平方根の考え（x²＝数 の形）。x＝±8（(±8)²＝64 ✓）

  2. 因数分解（和が1・積が−20の組: 5と−4）。(x＋5)(x−4)＝0 → x＝−5、x＝4（代入: 25−5−20＝0 ✓、16＋4−20＝0 ✓）

  3. 解の公式（因数分解が見えず、x²の係数も2）。a＝2、b＝−3、c＝−4。−b＝3、√の中: 9＋32＝41。分母 2a＝4。x＝(3±√41)/4 （代入: x＝(3±√41)/4 のとき x²＝(50±6√41)/16＝(25±3√41)/8、2x²＝(25±3√41)/4、−3x＝(−9∓3√41)/4、和は (25−9)/4＝4。よって 2x²−3x−4＝4−4＝0 ✓）

  4. 平方根の考え・かたまりのまま（展開したら損）。x＋2＝±√18＝±3√2 → x＝−2±3√2（(±3√2)²＝18 ✓）

B. つくる（着目した数量の言い方は一例）

  1. x²＝7x＋30（着目: xを2乗した結果。①x² ②7倍より30大きい数 7x＋30）

  2. x(2x＋1)＝10（着目: 花だんの面積。①縦×横＝x(2x＋1) ②10）

C. つくって、解いて、吟味する

  1. ① 真ん中の数をxとおく（xは正の整数）。3つの整数は x−1、x、x＋1。 ② 着目: いちばん小さい数といちばん大きい数の積。(x−1)(x＋1) と 3x＋3。 ③ (x−1)(x＋1)＝3x＋3 → x²−1＝3x＋3 → x²−3x−4＝0 → (x−4)(x＋1)＝0 → x＝4、x＝−1。 ④ xは正の整数（しかも x−1 も正）だから x＝−1 は不適。x＝4 を戻して: 3つの整数は3、4、5。3×5＝15、3×4＋3＝15 ✓。答え 3、4、5

  2. まちがい: xが0の可能性があるのに両辺をxで割った（L01 stretch・L05の急所の総復習）。割った瞬間に x＝0 が失われる。 正しくは移項して x²−6x＝0 → x(x−6)＝0 → x＝0、x＝6（代入: 0²＝6×0＝0 ✓、6²＝36＝6×6 ✓）

stretch

S1 調べ学習の導線のため、決まった解答はない。検索して見つけた説明を、本文L04の対比表（数字の例と一般の式の左右対比）と見比べて、「どの行が同じ操作か」を指させたら、この章の理解は本物だ。
