L05 いろいろな連立方程式

練習

  1. (1) ①を展開: 3x−3y＋2y＝3 → 3x−y＝3。y＝3x−3を②に代入: x＋2(3x−3)＝8 → 7x−6＝8 → x＝2、y＝3。(2, 3) 検算（元の式で）: ① 3(2−3)＋2×3＝−3＋6＝3 ✓、② 2＋6＝8 ✓ (2) ①×10: 3x＋y＝10。②と辺々を加える: 5x＝15 → x＝3、②に代入 6−y＝5 → y＝1。(3, 1) 検算（元の式で）: ① 0.9＋0.1＝1 ✓、② 6−1＝5 ✓ (3) ①×6: 2x＋3y＝18。②と辺々を加える: 4x＝12 → x＝3、①'に代入 6＋3y＝18 → y＝4。(3, 4) 検算（元の式で）: ① 3/3＋4/2＝1＋2＝3 ✓、② 6−12＝−6 ✓

  2. 「3x＋y＝10」と「x＋2y＝10」に分解。①から y＝10−3x を②に代入: x＋2(10−3x)＝10 → −5x＝−10 → x＝2、y＝4。(2, 4) 検算: 3×2＋4＝10 ✓、2＋2×4＝10 ✓（両方とも10——A＝B＝Cの3つの値がそろっていることまで確認できた）

  3. ①×10: 2x＋y＝7。②を展開: 3x−3y＋5y＝12 → 3x＋2y＝12。①'から y＝7−2x を②'に代入: 3x＋2(7−2x)＝12 → −x＋14＝12 → x＝2、y＝3。(2, 3) 検算（元の式で）: ① 0.4＋0.3＝0.7 ✓、② 3(2−3)＋5×3＝−3＋15＝12 ✓

stretch

S1 別の組み方の例:「A＝B」と「A＝C」→ { 3x＋y＝x＋2y（整理して y＝2x）, 3x＋y＝10 }。y＝2xを代入: 3x＋2x＝10 → x＝2、y＝4——同じ (2, 4)（検算: 6＋4＝10 ✓、2＋8＝10 ✓）。 楽さの比較例: 「A＝C と B＝C」（右辺が数の10になる組み方）は最初から基本の形なので整理いらずで楽。「A＝B」は移項の整理が1回増える。どれでも解は同じだから、数だけの辺（C）を使う組み方を選ぶのが定石。

L06 二つの文字で場面を表す——個数と代金

練習

  1. 大人をx人、子どもをy人とする。 人数: x＋y＝8　料金: 500x＋200y＝2500 ②÷100: 5x＋2y＝25。①×2: 2x＋2y＝16。引いて 3x＝9 → x＝3、①より y＝5。 場面に戻す: 3＋5＝8人 ✓、500×3＋200×5＝1500＋1000＝2500円 ✓（人数はどちらも自然数） 答え: 大人3人、子ども5人

  2. 30gのおもりをx個、50gのおもりをy個とする。 個数: x＋y＝12　重さ: 30x＋50y＝460 ②÷10: 3x＋5y＝46。①×3: 3x＋3y＝36。引いて 2y＝10 → y＝5、①より x＝7。 場面に戻す: 7＋5＝12個 ✓、30×7＋50×5＝210＋250＝460g ✓ 答え: 30gのおもり7個、50gのおもり5個

  3. 大きい方をx、小さい方をyとする。 和: x＋y＝26　関係: x＝3y＋2 ②を①に代入: 3y＋2＋y＝26 → 4y＝24 → y＝6、x＝20。 確かめ: 20＋6＝26 ✓、3×6＋2＝20 ✓ 答え: 大きい方20、小さい方6

  4. 説明の例: ① 大人をx人・子どもをy人として、人数の合計から x＋y＝8、料金の合計から 500x＋200y＝2500 の2本の式を作る。 ② この2本を連立方程式として解く。 ③ 解のxの値が大人の人数、yの値が子どもの人数である。 （3点のどれが欠けても説明として不足——特に②③を落とした「連立方程式にすればよい」だけでは説明にならない）

stretch

S1 一元一次方程式で: 30gのおもりをx個とすると50gは(12−x)個。30x＋50(12−x)＝460 → 30x＋600−50x＝460 → −20x＝−140 → x＝7。50gは5個——連立方程式の解と一致 ✓。 比べ方の例: 一元一次では「50gの個数を12−xと表す工夫」が最初に必要。連立なら見えている数量（個数の合計・重さの合計)をそのまま2本の式にできる。「工夫を1つ減らせるかわりに、式が1本増える」——これが二つの文字を使うよさの正体。

L07 速さ・割合の場面

練習

  1. 自転車をx分、徒歩をy分とする。 時間: x＋y＝13　道のり: 200x＋50y＝2000 ②÷50: 4x＋y＝40。②'−①: 3x＝27 → x＝9、①より y＝4。 場面に戻す: 9＋4＝13分 ✓、200×9＋50×4＝1800＋200＝2000m ✓ 答え: 自転車9分、徒歩4分

  2. 先月のAをx個、Bをy個とする。 先月: x＋y＝300　今月: 1.2x＋1.1y＝340 ②×10: 12x＋11y＝3400。①×11: 11x＋11y＝3300。引いて x＝100、①より y＝200。 場面に戻す: 100＋200＝300個 ✓、今月は 120＋220＝340個 ✓ 答え: 先月のA 100個、B 200個

  3. 上りの道のりをx m、下りの道のりをy mとする。 道のり: x＋y＝3000　時間: x/60＋y/90＝40 ②×180: 3x＋2y＝7200。①×2: 2x＋2y＝6000。引いて x＝1200、①より y＝1800。 場面に戻す: 1200＋1800＝3000m ✓、時間 1200/60＋1800/90＝20＋20＝40分 ✓ 答え: 上り1200m、下り1800m

  4. 説明の例: ① 先月のAをx個・Bをy個として、先月の合計から x＋y＝300、今月の合計（Aは1.2倍、Bは1.1倍になる）から 1.2x＋1.1y＝340 の2本の式を作る。 ② この2本を連立方程式として解く。 ③ 解のxの値が先月のAの販売数である（yではないことに注意）。

  5. 今月のAは先月の20%増だから 100×1.2＝120個。（解の100をそのまま答えないこと——問いが変われば、解から一歩計算した量が答えになる）

stretch

S1 Xの原価をx円、Yの原価をy円とする。 原価: x＋y＝3000　定価: 1.2x＋1.3y＝3700 ②×10: 12x＋13y＝37000。①×12: 12x＋12y＝36000。引いて y＝1000、①より x＝2000。 場面に戻す: 2000＋1000＝3000円 ✓、定価 2400＋1300＝3700円 ✓ 答え: Xの原価2000円、Yの原価1000円

L08 章末まとめ

3部屋の自己チェック（答えの例）

  • 部屋1: xとyの両方が決まって初めて式の成否が決まるから、解は値の組。二元一次方程式の解は一つとは限らない（変域しだいで無数）。連立方程式の解＝二つの方程式を同時に満たす組。確かめ方＝両方の式に代入して両方成り立つか見る。

  • 部屋2: 共通の作戦＝「文字を1つ消して、一元一次方程式に戻す（帰着）」。辺々を引くときは右辺どうしも引く。y＝…（x＝…）の形がある・作りやすいときは代入法が速い。

  • 部屋3: 4ステップ＝宣言→式2本→解く→場面に戻す。説明の3点＝①どの数量関係から式を2本作るか ②連立方程式として解く ③どの値が答えか。解がそのまま答えにならないのは、問いの量が解から計算する量（今年の人数・定価など）のとき。

総合演習

  1. (1) 理由例:「係数がそろえやすいので加減法」。②×3: 6x−3y＝9。①＋②×3: 10x＝10 → x＝1、②に代入 2−y＝3 → y＝−1。(1, −1)（検算: 4−3＝1 ✓、2＋1＝3 ✓） (2) 理由例:「y＝…の形があるので代入法」。①を②に代入: 3x−2(2x−3)＝4 → −x＋6＝4 → x＝2、①より y＝1。(2, 1)（検算: 1＝4−3 ✓、6−2＝4 ✓） (3) ①×10: 2x＋5y＝16。②から x＝2y−1 を代入: 2(2y−1)＋5y＝16 → 9y＝18 → y＝2、x＝3。(3, 2)（検算（元の式で）: 0.6＋1.0＝1.6 ✓、3−4＝−1 ✓）

  2. 50円カードをx枚、80円カードをy枚とする。 枚数: x＋y＝15　代金: 50x＋80y＝990 ②÷10: 5x＋8y＝99。①×5: 5x＋5y＝75。引いて 3y＝24 → y＝8、①より x＝7。 場面に戻す: 7＋8＝15枚 ✓、50×7＋80×8＝350＋640＝990円 ✓ 答え: 50円カード7枚、80円カード8枚

  3. ① 2つの数をx, y（x＞y）として、和から x＋y＝20、差から x−y＝6 の2本の式を作る。② この2本を連立方程式として解く。③ 解のxが大きい方、yが小さい方の数である。 実際に解く: ①＋②: 2x＝26 → x＝13、①より y＝7。13と7（検算: 13＋7＝20 ✓、13−7＝6 ✓）

  4. 作り方の例: x＝2, y＝3を先に決めて、左辺を適当に作り右辺を計算する。たとえば x＋y＝5、2x−y＝1。 確かめ: 2＋3＝5 ✓、4−3＝1 ✓——(2, 3)を解にもつ。（左辺の作り方はかなり自由。ただし一方の式が他方の何倍かになっていると解が1組に絞れない——S1参照）

stretch

S1 x＋y＝5 の自然数解: (1, 4), (2, 3), (3, 2), (4, 1)。2x＋2y＝10 の自然数解も (1, 4), (2, 3), (3, 2), (4, 1)——まったく同じ（2本目は1本目の両辺を2倍しただけなので、成り立つ組も完全に同じ）。 考えたことの例: 「2本目が新しい条件になっていないから、共通の組は1つに絞れない。条件が2つあるように見えて、実は1つしかない」。——連立方程式の解が1組に決まるには、中身のちがう条件が2本そろっていることが必要なのだと分かる（この「気づき」の先は、次章以降でゆっくり回収される）。
