L08 等式の変形——目的に合わせて式をつくり直す

練習

  1. (1) x＋3y＝9 → x＝9−3y（戻し検算: (9−3y)＋3y＝9 ✓） (2) 4x−y＝7 → −y＝7−4x → y＝4x−7（両辺に −1 をかけると全項の符号が反転。戻し: 4x−(4x−7)＝7 ✓） (3) V＝abc → c＝V/(ab)（ab≠0。戻し: ab×V/(ab)＝V ✓） (4) ℓ＝2(a＋b) → ℓ/2＝a＋b → a＝ℓ/2−b（戻し: 2(ℓ/2−b＋b)＝ℓ ✓）

  2. 5x＋2y＝10 → 2y＝10−5x → y＝5−(5/2)x（＝(10−5x)/2） 検算: x＝2 のとき y＝5−5＝0。元の等式に代入: 5×2＋2×0＝10 ✓ 成立。 （10 の割り忘れで y＝10−(5/2)x とすると、x＝2 で y＝5、5×2＋2×5＝20≠10 ✗ と即座にばれる）

  3. 誤り: 両辺を3でわるとき、右辺の 12 だけをわって −6b をわり忘れた（12/3＝4 は正しいが、−6b/3＝−2b の処理が抜けて −6b のまま残っている——本文主概念2と同型の「一部だけわる」事故）。正しくは 3a＝12−6b → a＝(12−6b)/3＝a＝4−2b。 検算: b＝1 のとき a＝2。元の等式: 3×2＋6×1＝12 ✓（誤答 a＝4−6b だと b＝1 で a＝−2、3×(−2)＋6＝0≠12 ✗）

  4. m＝(a＋b)/2 → 2m＝a＋b → b＝2m−a 検算: a＝60、m＝70 とすると b＝140−60＝80。平均 (60＋80)/2＝70 ✓（数学60点で平均70点にしたければもう1教科80点、と一発で読める式になった）

stretch

S1 h について: S＝(1/2)(a＋b)h → 2S＝(a＋b)h → h＝2S/(a＋b)（a＋b≠0。戻し: (1/2)(a＋b)×2S/(a＋b)＝S ✓） a について: 2S＝(a＋b)h → 2S/h＝a＋b → a＝2S/h−b かたまり視点が効いたのは、(a＋b) を1つの文字のように見て「(a＋b)＝」の形をまず作ったところ（2S＝(a＋b)h を (a＋b) について解いてから、b を移項）。かっこを1個の部品として扱う——L03で身につけた見方が、変形でも主役になる。

L09 式を読む——変形済みの式が教えてくれること

練習

  1. (1) 2(m＋n)−1: m＋n は整数だから 2×（整数）−1 の形。奇数を表す（2(m＋n−1)＋1 と書き直せば 2□＋1 の形が直接見える。m＝n＝1 → 3 ✓） (2) 5n＋5＝5(n＋1): n＋1 は整数だから 5の倍数（n＝2 → 15＝5×3 ✓） (3) 10a＋5＝5(2a＋1)（a は0以上の整数）: 2a＋1 は整数だから 5の倍数。さらに 2a＋1 は奇数だから「奇数×5」、つまり一の位が5の数と読める（a＝3 → 35 ✓。a が負の数も許すと −5 など「一の位が5」と言いにくい数が混ざるため、範囲の宣言が効いている）

  2. 外側レーンの半円の半径は r＋2 になるから、 差＝{2d＋2π(r＋2)}−(2d＋2πr)＝2π(r＋2)−2πr＝4π (m) 読みの一文: 「答えに d も r も残っていないので、1周の差は直線部分の長さにも半径にも関係なく、レーン幅2mならいつでも 4π m である。」（レーン幅1mの2倍の幅で、差もちょうど2倍——次のS1への伏線）

  3. 偶数と読める形: 4n＋6＝2(2n＋3)——2n＋3 は整数だから 2×（整数）＝偶数。 4でわると2余ると読める形: 4n＋6＝4(n＋1)＋2——4×（整数）＋2 の形。 n＝1 で確かめ: 4＋6＝10。10＝2×5 で偶数 ✓、10÷4＝2 余り 2 ✓。同じ数から2つの事実が読めた。

stretch

S1 レーン幅 a m のとき、外側の半円の半径は r＋a。 差＝{2d＋2π(r＋a)}−(2d＋2πr)＝2πa (m) 消えた文字: d と r（トラックの形・大きさは無関係）。残った文字: a（差を決めるのはレーン幅だけ）。 外側 k 本目のレーンの半径は r＋ka だから、1周は 2d＋2π(r＋ka)＝2d＋2πr＋2πka。内側との差は 2πka——k に比例して増える。だから「1本外に出るごとに 2πa ずつ」スタートラインを前にずらせば、全レーンの走る距離がそろう。（検算: k＝1 で 2πa、k＝2 で 4πa——練習2の結果（a＝2、k＝1 で 4π）とも一致 ✓）

L10 章末まとめ——文字の力の総点検

総合演習

  1. (1) 3(2x−y)−(x−4y)＝6x−3y−x＋4y＝5x＋y （検算: x＝1、y＝1 → 元 3×1−(−3)＝6、答 5＋1＝6 ✓） (2) (−6a²b)÷3ab×(−2b)＝(−2a)×(−2b)＝4ab （検算: a＝1、b＝1 → 元 (−6)÷3×(−2)＝4、答 4 ✓）

  2. 12ab²÷6b＝12ab²/6b＝2ab。a＝2、b＝−3 を代入して 2×2×(−3)＝−12 （直接代入: 12×2×9÷(6×(−3))＝216÷(−18)＝−12 ✓ 一致）

  3. 4ステップ:

    • 表す: n を整数とすると、連続する2つの奇数は 2n＋1、2n＋3 と表せる。

    • 計算・変形: (2n＋1)＋(2n＋3)＝4n＋4＝4(n＋1)

    • 読む: n＋1 は整数だから、4(n＋1) は 4×（整数）の形、つまり4の倍数を表す。

    • 結論: よって連続する2つの奇数の和は4の倍数になる。

    （検算: 5＋7＝12＝4×3（n＝2 のとき n＋1＝3）✓ 文字が1つでよいのは「連続する」の関係で2数目が自動的に決まるから）

  4. S＝(1/2)(a＋b)h → 2S＝(a＋b)h → h＝2S/(a＋b)（戻し検算: (1/2)(a＋b)×2S/(a＋b)＝S ✓） 便利な場面の例: 「面積と上底・下底が分かっている台形の高さを何度も求めるとき、毎回方程式を解かずに代入一発で出せる。」

  5. 7n＋7＝7(n＋1)。n＋1 は整数だから 7×（整数）の形、つまり7の倍数（n＝2 → 21＝7×3 ✓）

stretch

S1 （自作予想の一例——これ以外でも4ステップがそろっていればよい） 予想の例: 「連続する4つの整数の和は4の倍数になる？」 実験: 1＋2＋3＋4＝10、2＋3＋4＋5＝14、3＋4＋5＋6＝18——どれも4の倍数ではない。予想は反例で崩れた。では何者か？

  • 表す: n を整数とすると、連続する4つの整数は n、n＋1、n＋2、n＋3 と表せる。

  • 計算・変形: n＋(n＋1)＋(n＋2)＋(n＋3)＝4n＋6＝2(2n＋3)

  • 読む: 2n＋3 は整数だから 2×（整数）＝偶数。しかも 4n＋6＝4(n＋1)＋2 は 4×（整数）＋2 の形だから、4でわると必ず2余る——4の倍数には決してならない。

  • 結論: 連続する4つの整数の和は、偶数だが4の倍数ではない（4でわると2余る）。

（検算: 10÷4＝2余り2、14÷4＝3余り2 ✓）「予想が外れる→外れ方まで式が説明してくれる」——実験と文字式の役割分担が一番よく見える結末。
