L05 事象を文字式で表す——代入で確かめる

練習

  1. n を整数として、 (1) 5の倍数: 5n（この問題では n は自然数。n＝1、2、3 → 5、10、15 ✓） (2) 奇数: 2n＋1（2n−1 でもよい。n＝3 → 7 ✓） (3) 連続する2つの整数: n、n＋1（n＝4 → 4 と 5 ✓） (4) 3でわると1余る数: 3n＋1（n＝2 → 7。7÷3＝2余り1 ✓）

  2. (1) 10a＋b（a＝3、b＝7 → 37 ✓。「ab」は a×b＝21 になってしまうので不可） (2) 10b＋a（a＝3、b＝7 → 73 ✓ たしかに入れかわっている）

  3. n＝4 のとき 2n＋1＝9、2n＋3＝11。9 と 11 はどちらも奇数で、9 の次の奇数がちょうど 11（間の10は偶数）。よって「連続する2つの奇数」になっている ✓（もう1つ、n＝0 でも 1 と 3 で確認できる）

  4. n＝3（奇数）を代入すると、n と n＋2 は 3 と 5 になり、偶数ではない。この表し方は「差が2」は満たすが「偶数である」ことが式の形で保証されていない。偶数であることを保証するには 2n と 2n＋2（n は整数）とする必要がある。（n＝4 なら 4 と 6 でたまたま成功する——1つの代入で成功しても安心できないことも、この問題の教訓）

stretch

S1 4n＋3（n は整数）。 テスト: n＝0 → 3（3÷4＝0余り3 ✓）、n＝1 → 7（7÷4＝1余り3 ✓）、n＝2 → 11（11÷4＝2余り3 ✓）。 4n の部分が「4でわり切れる部分」、＋3 が「余り」を式の形で直接表している。

L06 式による説明①——4ステップの足場

足場つきの練習

足場1（穴埋め） 順に 2n／2n／m＋n。 完成形: m、n を整数とすると、二つの偶数は 2m、2n と表せる。和は 2m＋2n＝2(m＋n)。m＋n は整数だから、これは偶数である。よって二つの偶数の和は偶数になる。 （検算: m＝2、n＝3 → 4＋6＝10＝2×5 ✓）

足場2（根拠の選択） ア。 イは「1例の実験」であり、無限個ある場合すべてを説明できない。ウは説明したいことをそのまま言い直しただけで、根拠になっていない（循環）。「2×（整数）の形をしている」だけが、式の形を根拠にした説明になる。

足場3（部分記述）

  • 表す: m、n を整数とすると、偶数は 2m、奇数は 2n＋1 と表せる。

  • 計算・変形: 2m＋(2n＋1)＝2m＋2n＋1＝2(m＋n)＋1

  • 読む: m＋n は整数だから、2(m＋n)＋1 は 2×（整数）＋1 の形、つまり奇数を表す。

  • 結論: よって偶数と奇数の和は奇数になる。

（検算: m＝3、n＝2 → 6＋5＝11 奇数 ✓）

練習

  1. （足場1〜3の完成は上記のとおり）

  2. 4ステップ:

    • 表す: n を整数とすると、連続する2つの整数は n、n＋1 と表せる。

    • 計算・変形: n＋(n＋1)＝2n＋1

    • 読む: n は整数だから、2n＋1 は 2×（整数）＋1 の形、つまり奇数を表す。

    • 結論: よって連続する2つの整数の和は奇数になる。

    文字が1つでよい理由: 2つの整数に「連続する」という関係があり、片方を n と決めればもう片方は n＋1 と自動的に決まるから（自由に選べる数は1つだけ）。 （検算: n＝5 → 5＋6＝11 奇数 ✓）

  3. 3例の実験しかしておらず、無限にある奇数の組すべてについては何も示せていない点が不十分（どれだけ例を増やしても「試していない組」が残る）。すべての場合を尽くすには、二つの奇数を 2m＋1、2n＋1（m、n は整数）と文字で表して計算する必要がある——実験は予想を立てる道具、文字式は予想を確定させる道具、と役割が違う。

stretch

S1 4ステップ:

  • 表す: n を整数とすると、連続する3つの整数は n−1、n、n＋1 と表せる。

  • 計算・変形: (n−1)＋n＋(n＋1)＝3n

  • 読む: n は整数だから、3n は 3×（整数）の形、つまり3の倍数を表す。

  • 結論: よって連続する3つの整数の和は3の倍数になる。

（検算: n＝5 → 4＋5＋6＝15＝3×5 ✓） 最初の整数を n と置くと n＋(n＋1)＋(n＋2)＝3n＋3＝3(n＋1) となり、結論は同じだが分配法則で3をくくり出す一手が余分に要る。真ん中を n にすると −1 と ＋1 が打ち消し合って一直線——どの数を n にするかで計算の楽さが変わるのは、置き方も設計の一部だという好例。

L07 式による説明②——自力記述へ

練習

  1. 4ステップ:

    • 表す: 十の位を a、一の位を b とすると（a、b はどちらも1から9までの整数、a＞b）、もとの数は 10a＋b、入れかえた数は 10b＋a と表せる。

    • 計算・変形: (10a＋b)−(10b＋a)＝10a＋b−10b−a＝9a−9b＝9(a−b)

    • 読む: a−b は整数だから、9(a−b) は 9×（整数）の形、つまり9の倍数を表す。

    • 結論: よって2けたの自然数から入れかえた数をひいた差は、9の倍数になる。

    （検算: 52−25＝27＝9×3。a−b＝5−2＝3 で 9×3 と一致 ✓ かっこ外しの −(10b＋a)＝−10b−a が急所）

  2. 4ステップ:

    • 表す: n を整数とすると、連続する5つの整数は n−2、n−1、n、n＋1、n＋2 と表せる。

    • 計算・変形: (n−2)＋(n−1)＋n＋(n＋1)＋(n＋2)＝5n（−2、−1、＋1、＋2 が打ち消し合う）

    • 読む: n は整数だから、5n は 5×（整数）の形、つまり5の倍数を表す。

    • 結論: よって連続する5つの整数の和は5の倍数になる。

    （検算: 3＋4＋5＋6＋7＝25＝5×5（n＝5）✓）

  3. 欠けているのは読み（式と結論をつなぐ一文）。式 2(m＋n)＋1 と結論はあるが、「なぜこの式なら奇数と言えるのか」が書かれていない。書き足す一文: 「m＋n は整数だから、2(m＋n)＋1 は 2×（整数）＋1 の形、つまり奇数を表す。」——この一文があって初めて、式が結論の根拠として働く。

stretch

S1 4ステップ:

  • 表す: m、n を整数とすると、偶数は 2m、奇数は 2n＋1 と表せる。

  • 計算・変形: 2m−(2n＋1)＝2m−2n−1。ここで −1＝−2＋1 と分解して、2m−2n−2＋1＝2(m−n−1)＋1

  • 読む: m−n−1 は整数だから、2(m−n−1)＋1 は 2×（整数）＋1 の形、つまり奇数を表す。

  • 結論: よって偶数から奇数をひいた差は奇数になる。

（検算: 10−3＝7 奇数 ✓ m＝5、n＝1 のとき 2(5−1−1)＋1＝7 ✓） 2m−2n−1 のままでは「2×（整数）＋1」の形が見えない——読みたい形（2□＋1）から逆算して、−1 を −2＋1 に組み替える一手が本問のねらい。 （別解: 2m−2n−1＝2(m−n)−1 とくくれば「2×（整数）−1 の形」で、これも奇数の表し方として正しい（L05で 2n−1 も奇数と認めたとおり）。この別解ならひと組み替えで済む。上の解答は、ゴールの形（2□＋1）を指定されたときに組み替える練習として取り組んでほしい。）
