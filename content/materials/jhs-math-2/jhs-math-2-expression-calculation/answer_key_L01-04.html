L01 中1の道具箱を開け直す——項の意味

準備運動（前提診断）

  1. 3x＋5x＝8x

  2. 7a−2−4a＋6＝3a＋4（3a と 4 はまとめられない——文字の部分が違う）

  3. −(2x−5)＝−2x＋5（−は中の全項に配る。−5 の符号反転を忘れない）

  4. 2×(−3)＋7＝−6＋7＝1

  5. 80a 円

練習

  1. (1) 5x と −7（符号ごと切り取る） (2) −3a、4、−a（−a の係数は −1）

  2. (1) 8x−3x＝5x（検算: x＝2 → 元 16−6＝10、答 10 ✓） (2) 6a−1−2a＋5＝4a＋4（a＝2 → 元 12−1−4＋5＝12、答 8＋4＝12 ✓） (3) (3x＋4)＋(2x−6)＝5x−2（x＝2 → 元 10＋(−2)＝8、答 10−2＝8 ✓） (4) (5x＋2)−(3x−1)＝5x＋2−3x＋1＝2x＋3（−(−1)＝＋1 の反転に注意）

  3. 誤り: 5x と 3 をまとめて 8x にしたところ。5x と 3 は文字の部分が違うのでまとめられない。正しくは 7x＋3−2x＝5x＋3（これ以上計算できない。検算: x＝2 → 元 14＋3−4＝13、5x＋3＝13 ✓、8x＝16 ✗）

  4. x＝2 のとき、元の式 (5×2＋2)−(3×2−1)＝12−5＝7。答えの式 2×2＋3＝7。一致した ✓（代入検算は間違い探しの検査。1つの数の一致は完全な証明ではないが、計算ミスはたいていここで見つかる——L01 guide参照）

stretch

S1 矛盾しない。「式を簡単にする」は、x がどんな数でも通用する形の整理で、同類項をまとめ終えた 2x＋2 は、この形としてはこれ以上まとめられない、という意味。「式の値を求める」は、x を1つの数に固定したときの計算。x＝3 と決めれば 8 という数になるが、それは x＝3 のときだけの値で、式そのものの姿ではない。「すべての x に通用する形の限界」と「特定の x での値」——2つの文は別のことを言っている。

L02 単項式・多項式・同類項——文字が2種類になると

練習

  1. (1) 4ab: 単項式・次数2（a と b の2個） (2) −x²: 単項式・次数2（x が2個。係数は −1） (3) 3a＋2b: 多項式・次数1（一次式） (4) x²−5x＋6: 多項式・次数2（二次式。項は x²、−5x、6）

  2. (1) 6x＋4y−2x＋3y＝4x＋7y（x＝2、y＝1 → 元 12＋4−4＋3＝15、答 8＋7＝15 ✓） (2) 5a−7b−a＋2b＝4a−5b（a＝2、b＝1 → 元 10−7−2＋2＝3、答 8−5＝3 ✓） (3) x²＋3x−4x²＋x＝−3x²＋4x（x² と x は同類項ではない。x＝2 → 元 4＋6−16＋2＝−4、答 −12＋8＝−4 ✓） (4) 2ab−5＋3ab＋1＝5ab−4（a＝1、b＝2 → 元 4−5＋6＋1＝6、答 10−4＝6 ✓）

  3. 前半の 4x−x＝3x は正しい（−x の係数 −1 を正しく処理できている）。誤りは3x＋3y を 6xy とまとめたところ——x と y は文字の部分が違うのでまとめられない。正しくは 4x＋3y−x＝3x＋3y（これで終わり。検算: x＝2、y＝1 → 元 8＋3−2＝9、3x＋3y＝9 ✓、6xy＝12 ✗）

  4. x＝2、y＝1 のとき、元の式 6×2＋4×1−2×2＋3×1＝12＋4−4＋3＝15。答えの式 4×2＋7×1＝15。一致 ✓

stretch

S1 2ab と 2ba は同類項。乗法はかける順序を入れかえてよい（交換法則）から、ba＝ab であり、文字の部分は同じ。ふつうはアルファベット順に ab とそろえて書く。いっぽう x²y と xy² は同類項ではない。x²y＝x×x×y、xy²＝x×y×y で、かけ合わされている文字の組が違う（x が2個か、y が2個か）。「文字の部分が同じ」とは、並べ替えて一致すること——順序の違いはよいが、個数の違いはだめ。（確かめ: x＝2、y＝3 で x²y＝12、xy²＝18——値も別 ✓）

L03 多項式の加法と減法

練習

  1. (1) (3a＋b)＋(2a−4b)＝5a−3b（a＝2、b＝1 → 元 7＋0＝7、答 10−3＝7 ✓） (2) (5x−2y)−(3x−7y)＝5x−2y−3x＋7y＝2x＋5y（x＝1、y＝1 → 元 3−(−4)＝7、答 2＋5＝7 ✓） (3) (x²＋2x)−(3x²−x)＝x²＋2x−3x²＋x＝−2x²＋3x（x＝2 → 元 8−10＝−2、答 −8＋6＝−2 ✓）

  2. (1) 4(2a−b)＝8a−4b (2) −2(3x−y＋1)＝−6x＋2y−2（−2 を3つの項全員に配る。＋1 への分配忘れが定番事故）

  3. (1) 4(2a−b)＋3(−a＋2b)＝8a−4b−3a＋6b＝5a＋2b（a＝2、b＝1 → 元 4×3＋3×0＝12、答 10＋2＝12 ✓） (2) 3(2a−5b)−2(4a−7b)＝6a−15b−8a＋14b＝−2a−b（−2×(−7b)＝＋14b の符号に注意）

  4. a＝2、b＝1 のとき、元の式 3(4−5)−2(8−7)＝−3−2＝−5。答えの式 −2×2−1＝−5。一致 ✓

stretch

S1 通分して分子をまとめる。 (x＋y)/2＋(x−y)/3＝3(x＋y)/6＋2(x−y)/6＝{3(x＋y)＋2(x−y)}/6＝(3x＋3y＋2x−2y)/6＝(5x＋y)/6 （検算: x＝1、y＝1 → 元 2/2＋0/3＝1、答 6/6＝1 ✓。分子のかっこを外すとき −y の符号を守るのが急所）

L04 単項式の乗法と除法

練習

  1. (1) 5x×3y＝15xy (2) (−2a)×(−7b)＝14ab（負×負＝正） (3) 4m×(−3m)＝−12m²（m×m＝m²） (4) (−2x)²×3y＝4x²×3y＝12x²y（(−2x)²＝(−2x)×(−2x)＝＋4x²）

  2. (1) 15ab÷3a＝15ab/3a＝5b（検算: 5b×3a＝15ab ✓＝わり算は逆のかけ算で戻して確かめられる） (2) −18x²÷6x＝−3x（−3x×6x＝−18x² ✓） (3) 8a²b÷(−4ab)＝−2a（−2a×(−4ab)＝8a²b ✓） (4) 9xy÷(3/4)x＝9xy÷(3x/4)＝9xy×4/(3x)＝36xy/3x＝12y（12y×(3x/4)＝9xy ✓）

  3. (1) 6a×2b÷3ab＝12ab/3ab＝4（文字が全部消えて数だけ残る） (2) 12x²y÷4x÷y＝12x²y/(4x×y)＝3x（÷が2回——4x も y も分母へ） (3) 2a²÷4ab×6b＝(2a²×6b)/4ab＝12a²b/4ab＝3a（×6b は分子、÷4ab だけ分母）

  4. 24x²y÷8x＝24x²y/8x＝3xy。x＝−2、y＝3 を代入して 3×(−2)×3＝−18 （直接代入でも確かめられる: 24×(−2)²×3÷(8×(−2))＝24×4×3÷(−16)＝288÷(−16)＝−18 ✓ 一致）

stretch

S1 正しい計算（本文の型）: 2a²÷4ab×6b＝(2a²×6b)/4ab＝3a。 誤った解釈: 2a²÷(4ab×6b)＝2a²/24ab²＝a/(12b²)。 まったく別の式になる（a＝1、b＝1 で前者は 3、後者は 1/12）。×と÷が並ぶ式は左から順に読み、÷のうしろに来た「1つの単項式」だけが分母に入る。あとから来た ×6b まで分母に巻き込むには、かっこで 4ab×6b を1つのかたまりに束ねる必要がある——かっこは「どこまでが÷の相手か」の境界線を引く記号でもある。
