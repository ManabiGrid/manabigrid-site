L06 「〜でない」確率——1から引くという近道

練習

  1. ①直接: 6の目が出ないのは1〜5の5通り → 5/6 ②1から引く: 1−（6の目が出る確率1/6）＝5/6。一致 ✓

  2. 「少なくとも1枚は裏」が起こらないのは「2枚とも表」（1/4）のときだけ。 1−1/4＝3/4（検算・直接数え: 4通り中（表裏）（裏表）（裏裏）の3通り → 3/4 ✓）

  3. 同じ目になるのは6通り（6/36＝1/6）だから、同じにならない確率は 1−6/36＝30/36＝5/6（検算・直接数え: 36−6＝30マス → 30/36 ✓）

  4. 絵札はジャック・クイーン・キングが4種類の組（マーク）ごとに1枚ずつで 3×4＝12枚。絵札を引く確率は12/52だから、絵札でない確率は 1−12/52＝40/52＝10/13（検算: 絵札でないカードは52−12＝40枚 → 40/52 ✓）

stretch

S1 ①あいこは2種類: ⑴3人が同じ手＝（G,G,G）（C,C,C）（P,P,P）の3通り。⑵3人がグー・チョキ・パーを1つずつ＝樹形図で列挙すると GCP, GPC, CGP, CPG, PGC, PCG の6通り（かけ算点検: Aの手3通り×Bの手（残り2）×Cの手（残り1）＝3×2×1＝6 ✓）。あいこ合計 3＋6＝9通り → 9/27＝1/3。 ②あいこにならない確率＝ 1−9/27＝18/27＝2/3。（検算: 27通りのうち「だれかが勝つ」のは2種類の手だけが出そろう場合で 27−9＝18通り ✓）

L07 くじ引きは何番目でも公平か

練習

  1. 6通り（L07主概念2の樹形図）のうち、AもBも当たらないのは（は1,は2）（は2,は1）の2通り → 2/6＝1/3（検算: Aが当たる2＋Bが当たる2＋どちらも当たらない2＝6 ✓）

  2. 戻す場合、Bが引くときも3本すべてが袋にある。Bの引き方は3通り（同様に確からしい）で当たりは1本 → 1/3。戻さない場合（1/3）と同じ。（ルールを変えてもBの当たりやすさは変わらない、という点まで書ければ完答。）

  3. くじは当・はの2本。引き方は（当,は）（は,当）の2通り（同様に確からしい）。 ①起こり得る場合は全部で2通りで、どれも同様に確からしい。 ②Aの当たる確率は（当,は）の1通りで1/2、Bの当たる確率は（は,当）の1通りで1/2。 ③確率が等しいから、このくじ引きは公平である。

  4. （反論の例）樹形図で全部の場合をかき切ると6通りあり、どれも同様に確からしい。先に引くAの当たる確率は2/6＝1/3、後に引くBの当たる確率も2/6＝1/3で等しい。つまり引く順番によって当たりやすさは変わらないので、「最後に引く人が有利」という主張は正しくない。（「残り物に福」は確率の根拠を持たないことばである、という指摘まで書ければ十分。）

stretch

S1 3本のくじ（当・は1・は2）をA→B→Cが引き切るので、場合は3人へのくじの配られ方＝6通り（当がAの並び2通り・Bの並び2通り・Cの並び2通り）。Cが当たるのは当が3番目の（は1,は2,当）（は2,は1,当）の2通り → 2/6＝1/3。Aも1/3、Bも1/3（当が1番目・2番目の場合がそれぞれ2通り）——3人とも1/3で、最後の人も不利ではない。

L08 「確率が◯だから」——判断を説明する型

練習

  1. 袋X: 6個中赤3個 → 3/6＝1/2。袋Y: 9個中赤4個 → 4/9。通分すると 1/2＝9/18、4/9＝8/18 で 9/18＞8/18。赤玉が出やすいのは袋X（赤玉の個数はYの方が多いが、割合はXの方が大きい）。

  2. 36マスの表で数えると、和が6以下＝1＋2＋3＋4＋5＝15通り、和が7以上＝36−15＝21通り（検算・直接数え: 和7が6・和8が5・和9が4・和10が3・和11が2・和12が1で計21 ✓）。 ①全部で36通り・同様に確からしい ②Aの勝つ確率は15/36、Bの勝つ確率は21/36 ③確率が等しくないから公平ではなく、Bさんが有利。

  3. 和が7（6通り）を引き分けにすると、Aの勝ち＝和6以下の15通り、Bの勝ち＝和8以上の15通り（検算: 15＋6＋15＝36 ✓）。勝負が決まる場合どうしで比べて15対15、確率が等しいから公平になっている。

  4. 4通り（樹形図）のうち、あなたの勝ち＝（表表）の1通りで1/4、わたしの勝ち＝（裏裏）の1通りで1/4。勝つ確率が等しいから、公平と言える（1枚ずつの2通り＝1/2は引き分けで、どちらの有利にもならない）。

stretch

S1 （設計例——これ以外でも、3行型の説明と公平化案がそろっていれば正解） 「大小2つのさいころを投げて、ぞろ目なら私の勝ち、和が5ならきみの勝ち、どちらでもなければ引き分けにしてやり直し。どちらも『6分の1っぽい』から公平だよ」——実は、ぞろ目は6通り（6/36）、和が5は(1,4)(2,3)(3,2)(4,1)の4通り（4/36。ぞろ目と和5の重なりはない）。①全部で36通り・同様に確からしい ②私の勝つ確率6/36、きみの勝つ確率4/36 ③等しくないから不公平（私が有利）。公平に直す例: きみの勝ちを「和が5または和が11」に広げると 4＋2＝6通りで6/36どうし——公平（検算: 和11は(5,6)(6,5)の2通り・ぞろ目との重なりなし ✓）。ゲームのルールは、勝ち以外の場合（引き分け・やり直し）の扱いまで決めて初めて完成する——自作するときはここまで書くこと。

L09 章末まとめ——意味・求め方・説明の総点検

総合演習

  1. 二次元の表（2×6＝12マス・同様に確からしい）で、「裏かつ3の倍数（3・6）」は(裏,3)(裏,6)の2通り → 2/12＝1/6

  2. 「少なくとも一方は5」が起こらないのは「両方とも5でない」とき。5でない目は各さいころ5通りだから 5×5＝25マス → 1−25/36＝11/36（検算・直接数え: 大が5の6マス＋小が5の6マス−両方5の1マス＝11マス → 11/36 ✓）

  3. (1) 1枚目4通り×2枚目3通り（引いたカードは戻さない）＝12通り（樹形図: 12,13,14,21,23,24,31,32,34,41,42,43） (2) 30以上になるのは十の位が3・4のとき: 31,32,34,41,42,43の6通り → 6/12＝1/2 (3) 偶数になるのは一の位が2・4のとき: 12,32,42,14,24,34の6通り → 6/12＝1/2

  4. くじを当1・当2・は1・は2・は3と区別する。（Aの引くくじ, Bの引くくじ）の組は 5×4＝20通りで、どれも同様に確からしい。 Aが当たり＝1本目が当1（2本目は残り4通り）＋1本目が当2（同4通り）＝8通り（かけ算点検: 2×4＝8 ✓） → 8/20＝2/5 Bが当たり＝2本目が当1・当2の場合で、当1が2本目（1本目は残り4通り）＋当2が2本目（同4通り）＝8通り → 8/20＝2/5 ①全部で20通り・同様に確からしい ②Aの当たる確率は2/5、Bの当たる確率も2/5 ③確率が等しいから、「先に引く方が有利」という主張は正しくない（このくじ引きは公平である）。 （検算: Aが当たる8＋Bだけ当たる…ではなく、Bの当たる8通りにはA・Bとも当たる場合もふくまれる。全数点検: 両方当たり=当どうしの並び2通り、Aだけ当たり=2×3=6通り、Bだけ当たり=3×2=6通り、両方はずれ=3×2=6通り、計2+6+6+6=20 ✓。Aの当たり=2+6=8 ✓、Bの当たり=2+6=8 ✓）

stretch

S1 和ごとのマス数（2→1, 3→2, 4→3, 5→4, 6→5, 7→6, 8→5, 9→4, 10→3, 11→2, 12→1、合計36）から12マスずつの3グループを作る。分け方の例: A＝{2, 6, 7}（1＋5＋6＝12）、B＝{3, 4, 5, 10}（2＋3＋4＋3＝12）、C＝{8, 9, 11, 12}（5＋4＋2＋1＝12）。各12/36＝1/3で3人とも勝つ確率が等しい公平なゲームになる（分け方は他にもある——自分の案は必ず合計12ずつになるか検算すること）。
