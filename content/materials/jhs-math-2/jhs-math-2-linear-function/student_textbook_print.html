この単元で見えるようにしたいこと

一次関数は、ひとことで言うと「はじめの量」と「一定のペース」を式にしたものです！

y = ax + b

a は変わるペース、b ははじめの量です。a と b は決まった数で、a は 0 ではありません。「ペース」「はじめの量」は場面で読むときの言い方で、a も b も負の数になることがあります。そのときも、a は「xが1増えるときのyの変わり方」、b は「x=0のときのy」という意味です。表、式、グラフは別々の道具に見えますが、同じ関係を別の形で表しています。

最初は、文字の名前を覚えることよりも、「どれだけずつ変わるか」と「どこから始まるか」を分けて見ていきましょう。

単元の見取り図

レッスンすることキーワード
L1比例に「はじめの量」を足すy=ax+b
L2増え方を数で表す変化の割合

L1〜L2で「式の意味」、L3〜L5で「式とグラフの行き来」、L6〜L7で「使い方」へ進みます。

L1 比例から一次関数へ

今日のねらい

比例 y=ax に「はじめからある量」を加えると、一次関数 y=ax+b になることを理解します。

導入場面

空の水そうに、1分あたり3Lずつ水を入れるとします。x分後の水の量をyLとすると、式は y=3x です。

では、はじめから2L入っている水そうならどうでしょう。1分で3Lずつ増えることは同じですが、スタートが0ではありません。

例

はじめ2L、毎分3L増える水そうでは、

y = 3x + 2

となります。3x は増えた分、+2 ははじめから入っていた分です。

考え方

比例では、xが0のときyも0でした。一次関数では、xが0のときのyが0とは限りません。

  • a: xが1増えるときにyがどれだけ変わるか

  • b: x=0のときのy

比例は、b=0 の一次関数と見ることもできます。

よくあるまちがい

y=3x+2 の「+2」を「毎分2Lずつ増える分」と読んでしまうまちがいがあります。+2 は時間がたっても変わらない「はじめの量」です。増えるペースは 3x の3だけが表しています。

たしかめ

① はじめ500円持っていて、毎日50円ずつ貯金します。x日後の金額をy円とすると、式は何ですか。

② はじめ6L入っている水そうに、毎分2Lずつ水を入れます。x分後の水の量をyLとして式を作り、5分後の水の量を求めましょう。

ふり返り

「増えるペース」と「はじめの量」を分けて見られましたか。

L2 変化の割合

今日のねらい

表の中の値と、値の増え方を区別し、変化の割合が a にあたることを理解します。

導入場面

y=3x+2 の表を見ます。

x0123
y25811

yの値は2、5、8、11と変わります。増加量は、5-2、8-5、11-8なので、どこでも3です。

ここで見るのは、yの値そのものではなく、yがどれだけ増えたかです。どこを比べればよいでしょうか？

例

変化の割合は、次のように考えます。

変化の割合 = yの増加量 ÷ xの増加量

つまり、y÷x ではなく、「yの増加量」を「xの増加量」で割ります。

この表では、xが1増えるたびにyが3増えるので、変化の割合は3です。

考え方

b はスタート量なので、どこから始まるかを決めます。一方、変化の割合は増え方です。だから y=3x+2 でも y=3x+10 でも、変化の割合は3です。

xが1ずつでなく、2や3ずつ増える表でも同じです。たとえばxが2から5まで増えるなら、xの増加量は3です。yの増加量を、この3で割ります。

よくあるまちがい

表のyの値をそのままxで割って（y÷x）、変化の割合としてしまうまちがいがあります。上の表で 8÷2=4 と計算しても、それは変化の割合ではありません。割るのは「増加量どうし」です。

なお、比例（b=0）のときだけは、xが0でなければ y÷x が a に一致します。中1でこの計算が使えたのはそのためで、b が0でない一次関数では一致しません。

たしかめ

① y=4x-1 で、xが2から5まで増えるとき、xの増加量、yの増加量、変化の割合を求めましょう。

② y=-2x+7 で、xが1から4まで増えるとき、yの増加量と変化の割合を求めましょう。

ふり返り

「yの値」そのものと「yの増えた量」を混ぜずに読めましたか。

L3 式からグラフへ

今日のねらい

一次関数の式から点を取り、グラフが直線になることを確認します。

導入場面

y=2x+1 で、xにいくつか値を入れてみます。

x0123
y1357

この点を座標平面に取ると、まっすぐ並びます。

このレッスンでは、式から表を作り、表から点を取る流れに集中しましょう。グラフから式を読むことは、次のレッスンで扱います。

例

b=1 なので、グラフはy軸の1のところを通ります。a=2 なので、xが1増えるとyは2増えます。

考え方

b は切片、a は傾きとしてグラフに表れます。

  • 切片: y軸と交わるところ

  • 傾き: 右へ進んだときの上がり方

この傾きは、L2で表から求めた変化の割合と同じ数です。「yの増加量 ÷ xの増加量」が、グラフでは直線の傾きとして見えています。

また、点を結んだ直線は、取った点の間だけで止めず、両側にのばしてかきます。

よくあるまちがい

切片を「x軸と交わるところ」と混ぜてしまうまちがいがあります。切片 b は「x=0のときのy」、つまりy軸の上の点です。グラフをかくときは、まずy軸上の (0, b) に点を打ちます。

たしかめ

① y=x+2 の表を作り、点を3つ取って直線を引きましょう。

② y=2x-3 の切片と傾きを言い、x=0、1、2のときのyを求めて表にしましょう。

ふり返り

式の a と b が、グラフのどこに見えるか説明できますか。

L4 グラフから式へ

今日のねらい

グラフから切片と傾きを読み取り、式を復元します。

導入場面

グラフがy軸の2を通り、右に1進むと上に3進むとします。このとき、切片と傾きは何になるでしょうか？

前のレッスンとは向きが逆です。ここでは、すでにあるグラフから、切片と傾きを読み取ります。

例

切片が2、傾きが3なら、式は

y = 3x + 2

です。

考え方

右に進むと下がる直線では、傾きは負になります。たとえば右に1進むと2下がるなら、傾きは -2 です。

読み取りやすい格子点を2つ選ぶと、傾きを計算しやすくなります。

たとえば、右へ3進む間に上へ2進むなら、傾きは 2÷3 = 2/3 です。

よくあるまちがい

右下がりの直線を見て、傾きを正の数で答えてしまうまちがいがあります。「右へ進んだとき、上がるなら正、下がるなら負」。まず上がるか下がるかを言葉にしてから、数を読みましょう。

たしかめ

① 点 (0,1) と (3,3) を通る直線の傾きを求め、式を作りましょう。

② 切片が4で、右に1進むと2下がる直線の式を作りましょう。

ふり返り

「上がる」「下がる」を、傾きの正負と結びつけられましたか。

L5 条件から式を決める

今日のねらい

傾きと1点、または2点から一次関数の式を求めます。

導入場面

一次関数 y=ax+b で決めるものは、a と b の2つです。だから、「傾きと1点」や「x座標がちがう2点」のように、a と b を決められる手がかりが2つあれば、式を決められます。

例1 傾きと1点

傾きが2で、点 (3,7) を通るとします。

y = 2x + b
7 = 2×3 + b
7 = 6 + b
b = 1

式は y=2x+1 です。

例2 2点

点 (1,5) と (4,11) を通るとします。

2点から傾きを求めるときは、2つの点のx座標が異なることを先に確認します。

a = (11-5) ÷ (4-1) = 6 ÷ 3 = 2

傾きは2です。あとは点 (1,5) を使うと、

5 = 2×1 + b
b = 3

式は y=2x+3 です。2つの点のどちらを使っても、同じ式になります！

考え方

2点から式を決める方法は、2点のx座標が異なる場合に使います。x座標が同じでyの値がちがう2点を両方通る y=ax+b の式は、どんな方法でも作れません（同じxに2つのyが対応してしまうため）。この場合は、この単元の y=ax+b の決定問題とは分けて考えます。

よくあるまちがい

x座標を確かめずに、いきなり傾きの計算を始めてしまうまちがいがあります。最初の一手は「2点のx座標がちがうことの確認」です。

たしかめ

① 点 (2,7) と (5,16) を通る一次関数の式を求めましょう。

② 傾きが -1 で、点 (4,1) を通る一次関数の式を求めましょう。

ふり返り

式を決めるために、まず何を求めればよいか言えますか。

L6 二元一次方程式とグラフ

今日のねらい

yについて解くと y=ax+b の形になる二元一次方程式を、グラフと連立方程式の解に結びつけます。

導入場面

2x+y=6 は、xとyを使った方程式です。yについて解くと、

y = -2x + 6

となり、一次関数のグラフとして表せます。

2つの式を同時に満たすxとyを探す考え方は、連立方程式の考え方とつながります。連立方程式をまだ習っていない場合も、「2つの式を同時に満たすxとyの組を探している」と読みかえれば、このレッスンは進められます。

例

もう1つ、x+2y=8 をyについて解きます。

2y = -x + 8
y = -(1/2)x + 4

2つのグラフをかくと、交わる点があります。その点は、2つの式のどちらにも当てはまるでしょうか？

考え方

交点は、2つの直線の両方に乗っている点です。つまり、2つの方程式のどちらにも当てはまります。

交点を探すときは、片方の式を y=... と見て、「もう一方の式でも同じ y になる x はどれか」と考えます。

このレッスンでは、yについて解くと y=ax+b の形になる方程式を考えます。

よくあるまちがい

交点を聞かれて x=3 のように x の値だけを答えてしまうまちがいがあります。交点は座標平面上の点なので、たとえば交点が点 (3, 1) なら、答えは (3, 1) のように x と y の組で言います。

たしかめ

① まず x+y=5 は、yについて見ると y=5-x と表せます。そのうえで、もう一方の式でも同じ y になる x を探します。

x+y=5 と y=2x-1 の交点の座標を求め、その点を2つの式に代入して確認しましょう。

② 3x+y=9 をyについて解き、傾きと切片を答えましょう。

ふり返り

交点の座標が「連立方程式の解」と言える理由を説明できますか。

L7 一次関数の利用

今日のねらい

現実に近い場面を一次関数としてモデル化し、式が使える範囲も考えます。

導入場面

深さ40cmの水そうがあります。はじめの水の深さは4cmです。水を入れると、水面の高さは1分あたり3cmずつ上がります。

例

x分後の水の深さをycmとすると、

y = 3x + 4

です。水そうの深さは40cmなので、

40 = 3x + 4
36 = 3x
x = 12

12分でいっぱいになります。したがって、この場面で式を使える範囲は

0 ≤ x ≤ 12

です。

これは「0分後から12分後まで」という意味です。このとき、yの値の範囲は 4 ≤ y ≤ 40 になります（yにも変域があります）。12分をこえるとどうなるでしょう？水そうから水があふれるので、この式だけで場面を表すことはできません。

考え方

式に入れれば計算できる値でも、場面として意味があるとは限りません。一次関数を使うときは、式だけでなく、使える範囲も考えます。

また、「水面は毎分ちょうど3cmずつ上がる」と考えるのは、現実の場面を単純にした見方（モデル化の仮定）です。現実の場面を一次関数で表すときは、いつもこうした「一定のペースが続くとみなす」仮定が入っています。

よくあるまちがい

「計算できる」と「場面で使える」を混ぜてしまうまちがいがあります。y=3x+4 に x=100 を入れれば計算はできますが、この水そうは12分で満水になるので、100分後の水の深さがこの式の値になることはありません。式がこの場面を表しているのは 0 ≤ x ≤ 12 の間だけです。答えを出したら、その値が場面に合っているか一度確かめます。

たしかめ

① AさんとBさんが、同じ道を同じ向きに進みます。AさんはBさんの20m前の地点から毎分60mで、BさんはAさんと同時に出発して毎分80mで進むとします。BさんがAさんに追いつくのは、出発してから何分後ですか。

ヒント: AさんとBさんの差は、1分ごとに 80-60=20m ずつ縮まります。

追加ヒント（自分で考えたあと）

式で見るなら、出発してからの時間を x 分、Bさんの出発地点から進んだ道のりを y m とすると、Aさんは y=60x+20、Bさんは y=80x と表せます。追いつくときは、2つの y が同じになります。

② 上の水そうの式 y=3x+4 で、5分後の水の深さを求めましょう。また、x=15 のとき、この式はこの場面を正しく表しますか。理由も答えましょう。

ふり返り

「式では計算できるけれど、場面では使えない値」を見分けられましたか。

単元のまとめ

一次関数は、表、式、グラフ、場面をつなぐ道具です。

  • 式を見ると、増え方とスタート量がわかる。

  • 表を見ると、変化の割合を確かめられる。

  • グラフを見ると、全体の傾向や交点が見える。

  • 場面を見ると、式が使える範囲が決まる。

この4つを行き来できると、一次関数はぐっと使いやすくなります！

用語ミニ整理

用語意味
一次関数y=ax+b（a、b は決まった数で、a は0でない）の形で表せる関係
変化の割合yの増加量 ÷ xの増加量
傾き右に1進んだときの上がり下がり（式の a）
切片y軸と交わるところのy（式の b、x=0のときのy）
交点2つのグラフが交わる点。座標 (x, y) の組で表す
変域変数のとりうる値の範囲。x だけでなく y にも変域がある（この単元では主に、場面で式を使える x の範囲を扱う）
