L05 動かしてできる立体〜回転体

練習

  1. (1) 長方形（1辺が軸）→ 円柱　(2) 直角三角形（直角をはさむ1辺が軸）→ 円錐　(3) 半円（直径が軸）→ 球。見取図は、軸を一点鎖線・輪郭が左右対称になっていればよい。

  2. 縦5cm・横3cmの長方形を、5cmの辺（縦）を軸として1回転させたもの。軸に垂直な3cmの辺が底面の半径に、軸の5cmが高さになる。

  3. 円錐の母線＝もとの直角三角形の斜辺（しゃへん）。軸との関係: 円柱の母線は軸と平行（長方形の向かい合う辺だから）、円錐の母線は軸に対して傾いていて、頂点で軸と交わる。

  4. 平行な2辺（上底と下底）を軸に垂直にし、上底・下底の両方と垂直な脚を軸に重ねて回す。できるのは、上下の円の半径が上底・下底の長さになる「バケツ形」の立体（上下の円の大きさが違う）。図は、軸＝垂直な脚・平行な2辺が水平、が示せていればよい。

  5. 回している半円は平面図形、できた球は立体。（回す前は平面の住人、跡は空間の住人＝相手はだれ？チェック②）

stretch

S1 軸に垂直な平面で切る: 円柱→底面と合同な円・円錐→円（頂点に近いほど小さい）。軸をふくむ平面で切る: 円柱→長方形（もとの長方形2枚分）・円錐→二等辺三角形（もとの直角三角形2枚分）。どちらも左右対称 ✓

L06 平面の上に写し取る〜見取図・展開図・投影図

練習

  1. 例: 「正面の面の横の辺」と「奥行き方向の辺」。実際はどちらも立方体の1辺で等しいのに、奥行きの辺の方が短くかかれる。直角に見えない角の例: 上の面の角（実際は90°だが、見取図では平行四辺形の角になり、90°に見えない）。指摘の場所は図によって違ってよい——「実際の値」と「図での見え方」の食い違いが言えていれば正解。

  2. 展開図: 三角形2枚（底面）＋長方形3枚（側面）。対応: 各長方形の横の長さ＝底面の三角形の対応する辺の長さ（3枚がそれぞれ別の辺に対応）・縦の長さ＝柱の高さ（3枚とも共通）。組み立てたとき重なる辺に同じ印がつけてあれば、なおよい。

  3. 「立面図が長方形・平面図が円」→ 円柱／「立面図が三角形・平面図が円」→ 円錐／「立面図も平面図も円」→ 球

  4. 平面図だけでは区別できない（どちらも同じ半径の円になる）。立面図を見れば区別できる（長方形なら円柱・三角形なら円錐）。

  5. 立面図: 二等辺三角形（正面から見た輪郭）。平面図: 正方形と、その対角線2本。真上から見ると、頂点は正方形の中央の点に写り、頂点から底面の4つの頂点へ向かう4本の辺が、正方形の対角線として写る。

stretch

S1 切り口が長方形になる例: 底面に垂直な平面で、上の面から下の面までまっすぐ切り抜く（ただし側面と平行にならない向きでもよいし、平行なら正方形＝長方形の特別な場合になる）。切り口の上下の辺は上面・下面の中にでき、左右の辺は側面の中に垂直にできるから、4つの角がすべて直角——長方形。図は、切る平面が上下の面を貫いていることが見えていればよい。

L07 円をもう一度〜πと扇形

準備運動

  1. 円周＝10×3.14＝31.4（cm）（直径×円周率）

  2. 面積＝5×5×3.14＝78.5（cm²）（半径×半径×円周率）

  3. 直径を使うのは円周の式（面積は半径×半径）

練習（すべてπの検算 ①π ②次数 ③単位 を通過済み）

  1. 円周＝2π×7＝14π（cm）・面積＝π×7²＝49π（cm²） ✓

  2. 90/360＝1/4。弧＝2π×4×1/4＝2π（cm）・面積＝π×4²×1/4＝4π（cm²） ✓

  3. 45/360＝1/8。弧＝2π×8×1/8＝2π（cm）・面積＝π×8²×1/8＝8π（cm²） ✓

  4. 半径3cmの円全体の円周＝6π（cm）。弧2πはその 2π/6π＝1/3 → 中心角＝360°×1/3＝120°。検算: 2π×3×(120/360)＝2π ✓

  5. 検出はπの検算の①（πがついていない）。「半径×半径」で止まっていて円周率を掛けていない。正: π×5²＝25π（cm²）（3.14で近似するなら78.5cm²）

stretch

S1 例題の扇形（r＝6・ℓ＝4π）で S＝(1/2)×4π×6＝12π。本文の答えと一致 ✓。導出: S＝πr²×a/360 の a/360 を、ℓ＝2πr×a/360 から a/360＝ℓ/(2πr) と表して代入すると、S＝πr²×ℓ/(2πr)＝(1/2)ℓr ■

L08 表面積〜展開図で平面にもどす

準備運動

  1. 円周＝6π（cm）・面積＝9π（cm²）

  2. 120/360＝1/3。π×9²×1/3＝27π（cm²）

  3. 底面の円周

練習（すべてπの検算を通過済み。円錐は展開図→逆向き手順で解く）

  1. 底面積＝5×5＝25（cm²）×2枚＝50。底面の周＝20（cm）→ 側面積＝20×8＝160（cm²）。表面積＝50＋160＝210（cm²）

  2. 底面積＝π×3²＝9π ×2枚＝18π。側面: 横＝円周6π・縦＝7 → 42π。表面積＝18π＋42π＝60π（cm²） ✓

  3. 底面積＝8×8＝64。側面積＝(1/2×8×6)×4＝24×4＝96。表面積＝64＋96＝160（cm²）

  4. (1) 底面の円周＝2π×2＝4π。母線6cmの円全体の円周＝12π。4π/12π＝1/3 → 中心角＝120° (2) 側面積＝π×6²×1/3＝12π。底面積＝π×2²＝4π。表面積＝12π＋4π＝16π（cm²） ✓

  5. 誤り: 扇形の半径に底面の半径4cmを入れている（正しくは母線8cm）。さらに中心角も未確認のまま180°としている。正しい手順: 底面の円周＝8π・母線8cmの円全体の円周＝16π → 8π/16π＝1/2 → 中心角180°（これは偶然一致）。側面積＝π×8²×1/2＝32π（cm²）。検算: πrℓ＝π×4×8＝32π ✓

stretch

S1 底面の円周＝2πr。母線ℓの円全体の円周＝2πℓ。何分のいくつ＝2πr/2πℓ＝r/ℓ。側面積＝πℓ²×(r/ℓ)＝πrℓ ■　検算: 練習4(2)は π×2×6＝12π——本編の答えと一致 ✓

検算記録（著者自己検査）: L07-2 2π・4π／L07-3 2π・8π／L08-2 60π／L08-4 16π／L08-5 32π はいずれも別経路（πrℓ・a/360の約分）で再計算し一致を確認。
