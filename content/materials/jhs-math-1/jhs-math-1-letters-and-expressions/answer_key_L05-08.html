L05 数量を文字式で表す②——割合・速さ・残り

練習

  1. (1) 基準量は x 円 → x×2/5＝(2/5)x（円）（0.4x 円でもよい。検算: x＝1000 → 400円。1000円の5分の2 ✓） (2) 基準量は a 人。60%＝0.6 → 0.6a（人）（検算: a＝100 → 60人 ✓） (3) 3割引き＝定価の7割 → 0.7y（円）（検算: y＝1000 → 700円。1000円の3割引き＝300円引き ✓） (4) 15%＝0.15 → 0.15b（kg）（検算: b＝100 → 15kg。100kg の 15% ✓）

  2. (1) 時間＝道のり÷速さ → x/5（時間）（検算: x＝10 → 2時間。10kmを時速5km ✓） (2) 秒速＝道のり÷時間 → a/80（m）（検算: a＝160 → 秒速2m ✓） (3) 2時間30分＝2.5時間。道のり＝速さ×時間 → 2.5v（km）（検算: v＝10 → 25km ✓）

  3. (1) 配った個数は 2個×7人＝14個 → 残り (a−14)個（検算は練習5で行う） (2) 代金は 6x 円 → おつり (1000−6x)円（検算: x＝100 → 1000−600＝400。100円のパン6個で600円、おつり400円 ✓）

  4. y＝200 のとき、200円の25%は 200×0.25＝50円のはず。y/25＝200/25＝8 で大きく不一致——誤り。y/25 は「y を 25 でわった数」で、25%（＝0.25倍）とは別もの。正しくは 0.25y（検算: 0.25×200＝50 ✓）。%は小数（または分数）に直してから、基準量にかける。

  5. a＝20 のとき、式の値は 20−14＝6。場面: 20個から14個配れば残り6個——一致 ✓

stretch

S1 p＝500: 0.8p＝400、p−0.2p＝500−100＝400 ✓。p＝1000: 800 と 1000−200＝800 ✓。p＝2500: 2000 と 2500−500＝2000 ✓——すべて一致。理由: 2割引きの値段は「定価の10割から2割を引いた残り」、つまり定価の8割。「10割−2割＝8割」という割合どうしの引き算が先に成り立っているから、どの p でも p−0.2p と 0.8p は同じ量になる。

L06 式の値——文字に数を入れて調べる

練習

  1. (1) 3×4−5＝12−5＝7　(2) 4/2＋1＝2＋1＝3　(3) 4²＝16

  2. a＝−2 をかっこ付きで代入する。 (1) 5×(−2)＋13＝−10＋13＝3 (2) −(−2)＝2（−a が正の数になる例） (3) (−2)²＝(−2)×(−2)＝4 (4) 3−2×(−2)＝3−(−4)＝3＋4＝7

  3. (1) 1000−120×3＝1000−360＝640。意味: ドーナツを3個買ったときのおつりが640円ということ。 (2) 1000−120×8＝1000−960＝40。意味: 8個買うと代金が960円になり、おつりは40円——1000円でぎりぎり8個買える、ということまで読み取れる。

  4. かっこ付きで代入すると、4−x＝4−(−5)＝4＋5＝9。「−1」は 4−5 と計算した誤り（−5 の符号を落として代入している）。正しい値は 9。

（補足: −x＝−(−5)＝5 と、−のついた式の値が正になることも、あわせて確認しておこう）

  1. x＝1/2 をかっこ付きで代入する。 (1) 4×(1/2)＋3＝2＋3＝5 (2) 2×(1/2)−3＝1−3＝−2（分数を入れても、手順は整数のときと同じ）

stretch

S1 代入して計算すると次の表になる。

x0500100015002000
15−0.006x1515−3＝1215−6＝915−9＝615−12＝3

x が 500 増えるごとに、式の値は 3 ずつ小さくなる（0.006×500＝3）。「同じだけ増えると、同じだけ減る」という規則正しい変化が、1本の式から読み取れる。

L07 式を読む——3a＋2はこれで答え

練習

  1. (1) 2a はおにぎり2個の代金、140 はお茶1本の値段 → 買い物全体の代金の合計（円） (2) 140 はお茶の値段、a はおにぎり1個の値段。その差 → お茶1本がおにぎり1個より何円高いか

  2. (1) x×4 ＝ 1辺の4倍 → 正方形のまわりの長さ（cm） (2) x×x ＝ 1辺×1辺 → 正方形の面積（cm²）

  3. 4n……4つの辺を、それぞれ n 個ずつと数えた石の総数。−4……そのように数えると4すみの角の石をそれぞれ2回ずつ数えてしまうので、二重に数えた分の4個を引いている。

  4. 支払った金額＝色紙10枚の代金−値引き → (10b−100)円。b＝50 でテスト: 式からは 10×50−100＝400。場面では、50円×10枚＝500円から100円引きで400円——一致 ✓

（10b−100 円のかっこについて: 差の式なので、単位をつけて書くときは (10b−100)円 とかっこでくくる）

  1. (1) 1000−80x に x＝5 → 1000−400＝600。(1000−80)x に x＝5 → 920×5＝4600 (2) 残金を表すのは 1000−80x（600円は場面の常識と合う）。(1000−80)x は、かっこがあるために 先に 1000−80＝920 を計算し、その 920 を x 倍する式になっている（920円の品物を x 個買う代金のような式）。値も 4600 と、持っている 1000円を超えてしまい、残金としてありえない。かっこがない 1000−80x では、かけ算 80x がひき算より先に計算されるから、「1000円から代金 80x 円を引いた残り」を正しく表せる。かっこの有無で、式の意味も値もまるごと変わる。

stretch

S1 n＝4: 4(n−1)＝4×3＝12、4n−4＝16−4＝12、2n＋2(n−2)＝8＋2×2＝12——一致 ✓。n＝10: 4×9＝36、40−4＝36、20＋2×8＝36——一致 ✓。（「どんな n でも」を代入なしで示す方法はL11〜L12で手に入る。L12練習5で実際にやる）

L08 等式——「等しい」を宣言する記号

練習

  1. (1) 代金 8x 円が 640円と等しい → 8x＝640 (2) 1人分 y/5 個が 7個と等しい → y/5＝7 (3) まわりの長さ 4a cm が b cm と等しい → 4a＝b

  2. ア。ア: 残り＝200−4x で、それが 40 → 200−4x＝40 ✓。イの場面をそのまま手順どおりに書き取ると 200−40＝4x で、式の形が違う（この等式も x の値のつながりとしては 200−4x＝40 と関係があるが、問われているのは「場面をそのまま書き取った式」）。ウは 200＋4x＝40 になってしまう。

  3. （例）x＝40 と想定する。左辺: 200−4×40＝200−160＝40。右辺: 40。両辺の値が一致する ✓（1枚4円の折り紙を40枚買うと160円で、200円から残るのは40円——場面とも合う）

  4. 残り＝はじめの長さ−切り取った長さ → a−b＝2

（2＝a−b でもよい。等号の左右は対等だから、どちらの向きでも同じ関係を表す）

stretch

S1 左辺 x＋3 の値: x＝5 → 8、x＝6 → 9、x＝7 → 10、x＝8 → 11。右辺 10 と等しくなるのは x＝7 のとき。この「等式を成り立たせる x を探す」作業こそ、次章「方程式を解く」の原型。次章では、代入で1つずつ試さなくても x にたどり着く方法を学ぶ。
