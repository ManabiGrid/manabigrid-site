L01 決めると、決まる——関数

練習

  1. (1) いえる。1辺を決めれば周りの長さは y＝4x でただ一つに決まる（x＝3なら y＝12 の一つだけ）。 (2) いえない。同じ年れいでも身長はいろいろあり、xを決めてもyがただ一つに決まらない。 (3) いえる。自然数を決めれば約数の個数はただ一つに決まる（x＝6なら約数は1, 2, 3, 6でy＝4）。 (4) いえない。面積が12cm²でも、たては1cm・2cm・3cm……といくらでもあり、ただ一つに決まらない。

  2. (1) いえる。y＝20x で、長さを決めれば重さはただ一つに決まる。 (2) いえる。重さyを一つ決めると、長さは x＝y÷20 のただ一つ（y＝60なら x＝3 の一つだけ。確かめ: 20×3＝60 ✓）。この例は両方向とも関数になる例——L01反例1・2（逆向きが関数にならない例）との対比になっている。

  3. ア・ウ。アは「時刻を決めれば気温はただ一つ」なので正しい。イは誤り——式に表すことができない関数関係もある（判定基準は「ただ一つ決まるか」だけ）。ウは正しい——同じ気温になる時刻が2回以上ありえる（朝と夕方など）ので、逆向きはただ一つに決まるとは限らない。

  4. (1) ×。逆向きは成り立つとは限らない。正しくは「yがxの関数であっても、xがyの関数であるとは限らない」（反例: L01の影の例——影の長さが96cmになる時刻が午前と午後の2つあった日には、長さを決めても時刻はただ一つに決まらない）。 (2) ○。例: 時刻と気温。式に書けなくても、ただ一つに決まっていれば関数である。

stretch

S1 yはxの関数といえる（整数を決めれば絶対値はただ一つ。x＝−3なら y＝3 の一つだけ）。xはyの関数といえない。y＝3になるxは 3と−3の2つあり、yを決めてもxがただ一つに決まらないから（反例1つで十分）。

L02 変数の動く範囲——変域

練習

  1. (1) 5 ≦ x ≦ 12　(2) −4 ＜ x ＜ 3　(3) 0 ≦ x ＜ 7（「7より小さい」は7をふくまないので＜）　(4) −1 ≦ x（右端なし）

  2. (1) 2の位置に●・6の位置に●、間を太線（両端ともふくむ）。 (2) −3の位置に○・1の位置に●、間を太線（−3はふくまず、1はふくむ）。

  3. 満水までの時間は 24÷3＝8（分）。よって xの変域: 0 ≦ x ≦ 8。深さは y＝3x だから、x＝0でy＝0、x＝8でy＝24（検算: 3×8＝24＝水そうの深さ ✓）。yの変域: 0 ≦ y ≦ 24。

  4. (1) ×。「未満（より小さい）」はその値をふくまないので、正しくは x ＜ 4（x ≦ 4 は「4以下」）。 (2) ○。−2以上2以下の範囲には、整数だけでなく0や1.5のような数もすべてふくまれる。

stretch

S1 3 ≦ x ≦ 3 は「3以上かつ3以下」だから、あてはまる値は x＝3 のただ一つ。範囲がつぶれて1点だけになる。いっぽう 3 ＜ x ＜ 3 は「3より大きく、かつ3より小さい」——そんな数は存在しない（あてはまる値なし）。不等号の「＝つき・＝なし」のちがいが、端の1点をふくむかどうかを分けている。

L03 比例を捉え直す——負の数の世界へ

練習

  1. ア: 商は 6÷1＝6、12÷2＝6、18÷3＝6、24÷4＝6——一定。比例・比例定数6（y＝6x）。 イ: 商は 10÷1＝10、8÷2＝4——一致しないので比例ではない（差が2ずつの表）。 ウ: 商は 10÷(−2)＝−5、5÷(−1)＝−5、(−5)÷1＝−5、(−10)÷2＝−5——一定。比例・比例定数−5（y＝−5x。負の数が混ざっていても商一定なら比例）。

  2. 完成した表:

    x−2−1012
    y840−4−8

    検算: y÷x は 8÷(−2)＝−4、(−8)÷2＝−4 ✓（x＝0の列は y＝−4×0＝0）。xが1増えるごとに、yは4ずつ減る。

  3. (1) 比例する・比例定数5。 (2) 比例する・比例定数1/3（y＝x/3＝(1/3)x。わり算の形でも y＝ax に直せれば比例）。 (3) 比例しない。x＝1でy＝3、x＝2でy＝4——商は3と2で一定にならない。 (4) 面積は y＝x×6÷2＝3x。比例する・比例定数3（検算: x＝4なら 4×6÷2＝12＝3×4 ✓）。

  4. (1) ×。比例定数が負のときはxが増えるとyは減る（例: y＝−3x）。正しくは「比例定数が正ならyは増え、負なら減る」。 (2) ○。y＝ax にx＝0を代入すると y＝0。

stretch

S1 y＝−3x の表では、xが1増えるとyは3ずつ減る（例: x＝1→2 でy＝−3→−6）。まとめの例: 「xが1増えたときのyの変わり方は比例定数のぶんだけで、比例定数が正なら増え、負なら減る」。

L04 平面の住所——座標

練習

  1. Ａ(2, 4)・Ｂ(−3, 2)・Ｃ(−4, −4)・Ｄ(3, −1)・Ｅ(0, −2)・Ｆ(5, 0)（Ｅはy軸上なのでx座標0、Ｆはx軸上なのでy座標0）。

  2. Ｐ＝原点から右へ4・上へ3／Ｑ＝左へ2・上へ5／Ｒ＝左へ5・下へ2／Ｓ＝右へ1・下へ4／Ｔ＝y軸上の高さ4／Ｕ＝x軸上の−3。（読み合わせ検算: とった各点から座標を読み直して元の組と一致することを確認 ✓）

  3. (1) ×。座標は「xが先、yが後」の約束なので、(2, 3)と(3, 2)は別の点（(2, 3)は右へ2・上へ3、(3, 2)は右へ3・上へ2）。 (2) ○。x軸上の点は上下方向のずれが0、つまりy座標が0。 (3) ○。原点は2本の数直線の0どうしの交点。

  4. a＝0（y軸上の点は左右方向のずれが0、つまりx座標が0）。bはどんな値でもよい。

stretch

S1 Ｂ(3, −2)（x軸をはさむと上下が反転＝y座標だけ符号が変わる）。Ｃ(−3, 2)（y軸をはさむと左右が反転＝x座標だけ符号が変わる）。Ｄ(−3, −2)（原点をはさむと両方反転＝両方の符号が変わる）。規則の例: 「対称の軸に垂直な方向の座標だけ符号が変わり、原点対称では両方変わる」。
